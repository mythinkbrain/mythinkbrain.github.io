6. Sebuah toko di MOG mencantumkan harga pada sebuah selimut yang dijualnya sebesar Rp. 200.000,00 dan tercantum juga diskon 15%. Harga selimut tersebut setelah didiskon adalah *​


⚠DIKETAHUI⚠

Harga awal = Rp. 200.000,-

Besar diskon = 15%

❔DITANYAKAN❔

Harga setelah didiskon = ❓

✏PENYELESAIAN✏

Misalkan :

Harga awal = a

Besar diskon = b

Harga setelah didiskon = c

Maka harga barang setelah didiskon dapat ditentukan dengan :

c = a - (ab)

c = 200.000 - (200.000 × 15%)

c = 200.000 - (200.000 × 15/100)

c = 200.000 - (2000 × 15)

c = 200.000 - 30.000

c = 170.000

✒KESIMPULAN✒

Maka harga barang tersebut setelah didiskon adalah Rp. 170.000,-

29. apakah artinya nama bonita


Arti nama "Bonita".

Pembahasan

Nama merupakan kata untuk memanggil seseorang yang biasanya memiliki arti atau makna tertentu. Nama akan diberikan kepada bayi berumur 7 bulan setelah dilahirkan.

Contoh kalimatnya :

• Nama saya adalah Bonita.

• Adik menamai Milky pada kucingnya.

• Aku menyayangimu atas nama Allah.

Sebenarnya, "Bonita" berasal dari kosakata bahasa Spanyol yang berarti imut, cantik membahana, indah, atau mempesona. Sehingga, "Bonita" adalah nama untuk anak perempuan.

Nama anak khas Spanyol :

Blanco = putih, bersih.Arcoíris = pelangi, warna-warni.Ligero = sinar, cahaya.Estrella = bagaikan bintang.Regalo = karunia dari Tuhan.Bonita = imut, cantik, indah.Floria = bunga mekar.Humberto = laki-laki cerdas.Alessandra = perempuan yang pemberani dan tangguh.

[tex]diskon = 15\% \times harga \: kotor \\ diskon = \frac{15}{100} \times 15.000.000 \\ diskon = 2.250.000[/tex]

Jawab :

harga bersih = harga kotor - diskon

harga bersih = Rp15.000.000 - Rp2.250.000

harga bersih = Rp12.750.000
