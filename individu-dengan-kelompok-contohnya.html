10. Berikan contoh Individu dengan individu, Individu dengan kelompok, Kelompok dengan kelompok!!!


Jawaban:

Individu = satu orang / sendiri

Kelompok = ada banyak orang yang menjadi satu.

(maka)

Contoh dari:

Individu - individu (satu orang ke satu orang) = Orang nanya jalan ke orang lain // orang sakit ketemu dokter // penyanyi ketemu produser // anak buah ketemu bosnya // Kamu ketemu teman kamu.Individu - kelompok (satu orang ke banyak orang) = Guru ngajar banyak murid // penyanyi bernyanyi didepan banyak orang // pembicara ngomong di depan banyak tamu undangan (di seminar) // ustad ceramah kepada banyak jemaah di masjid // pendeta berbicara didepan banyak jemaat gereja // Kamu bergabung di geng yang isinya teman-teman kamu.Kelompok - kelompok (banyak orang ke banyak orang) = Keluarga 1 bertemu keluarga 2 untuk berkumpul // Grup band tampil berkolaborasi dengan grup band lain (apapun genre musiknya) // Ibu-ibu dan keluarga besar menghadiri pengajian // Klub bola bertemu klub bola lain dalam sebuah pertandingan sepakbola.

Ketiganya termasuk "interaksi sosial" dimana orang ke satu atau lebih orang bertemu untuk berbicara dan bertukar pikiran melalui pertemuan itu.

24. Jelaskan dan contoh hubungan antara individu dengan kelompok ,individu dengan individu dan kelompok dengan kelompok?

Contoh : perlombaan sepak bola, tawuran antar kelompok.
