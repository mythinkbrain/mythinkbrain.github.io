1. Lampu listrik tertera 100 watt, 220 volt dipasang pada tegangan 110 volt, maka daya nyalanya ... watt


Penjelasan:

P = V²/R

P sebanding dengan V²

tegangan 220 menjadi 110 => ½ nya

maka daya menjadi (½)² × 100W

= ¼×100 = 25 watt

Daya nyala saat dipasang pada 110 V adalah 25 Watt.

Pendahuluan :

Daya listrik adalah besaran yang menyatakan kemampuan suatu alat listrik yang mengubah energi listrik menjadi bentuk energi lainnya. Satuan besarannya menurut SI adalah Watt ( P ) yang diambil dari nama ilmuwan Inggris yaitu James Watt

1 Watt = 1 J/s

Untuk menghitung besar daya yang dimiliki oleh suatu alat listrik digunakan

P = W/tPembahasan :

Diketahui :

Lampu listrik tertera 100 watt, 220 volt dipasang pada tegangan 110 volt

Ditanya :

Daya nyala saat dipasang pada tegangan 110 V?

Jawab :

V1²/P1 = V2²/P2

220²/100 = 110²/P2

48400/100 = 12100/P2

        4/100 = P2

           100 = 4.P2

      100 : 4 = P2

             25 = P2

Kesimpulan :

Jadi, daya nyala saat dipasang pada tegangan 110 V adalah 25 Watt.

3. dua lampu pijar masing masing 75 watt-220 volt dan 25 watt-220 volt dipasang seri pada potensial 100 volt. berapa daya tiap lampu ?


Diket :

P1 = 75 watt
P2 = 25 watt
V1 = 220 volt
V2= 100 volt
Seri


Dit : Daya tiap lampu jika dipasang seri pada potensial 100 volt.....?


Jwb :


# Cari hambatan tiap lampu


> hambatan pd lampu P1

R1 = V1^2 / P1
R1 = 220^2 / 75
R1 = 48.400 / 75
R1 = 645 ohm

> hambatan pd lampu P2

R2 = V1^2 / P2
R2 = 220 ^2 / 25
R2 = 48.400 / 25
R2 = 1.936 ohm

# Total hambatan seri

Rs = R1 + R2
Rs = 645 + 1.936
Rs = 2.581 ohm

# Lanjut Cari arus total seri pd teg. 100 volt

I = V2 / Rs
I = 100 / 2.581
I = 0,0387 A

> Krn seri Arus pd tiap lampu sama

I = I1 = I2 = 0,0387 A


# Cari Daya tiap lampu

> Daya lampu P1

P1 = I^2 . R1
P1 = 0,0387^2 . 645
P1 = 0,00149769 . 645
P1 = 0,966 w ( 1 watt)

> Daya lampu P2

P2 = I^2 . R2
P2 = 0,0387^2 . 1.936
P2 = 0,00149769 . 1.936
P2 = 2,899 w ( 3 watt)


Jadi daya tiap lampu jika dipasang seri pada potensial 100 volt adalah, lampu dg daya 75 watt akan menjadi 1 watt, sedang lampu dg daya 25 watt akan menjadi 3 watt

10. Pada sebuah lampu tertulis spesifikasi 100 watt, 220 volt. daya lampu tersebut jika dipasang pada tegangan 110 volt adalah....


ENERGI dan DAYA LISTRIK

V₁ = 220 V. P₁ = 100 W
V₂ = 110 V. P₂ = __?

Daya listrik
P = V² / R
R = V² / P

perbandingan
V₂² / P₂ = V₁² / P₁
P₂ = (V₂ / V₁)² P₁
P₂ = (110/220)² × 100 W
P₂ = ¼ × 100 W
P₂ = 25 W ✔️Diketahui :

P2....?

lampu sama, jadi dianggap hambatan atau R itu tetap.
langsung dibandingkan :
P1/V1² = P2/V2²
100/220² = P2/110²
100/48400 = P2/12100
P2 = (100 x 12100) / 48400
P2 = 25Watt

R = v * v / p
