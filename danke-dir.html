7. Mohon diberi pembahasannya juga ya kak. Danke


Jarak titik ke bidang pada suatu kubus adalah adalah jarak terpendek yang di tarik dari titik ke bidang dan harus tegak lurus dengan bidang tersebut . Pada kubus ABCD.EFGH dengan rusuk s, memiliki

12 diagonal sisi yaitu AC, BD, EG, HF, AH, ED, BG, FC, AF, BE, DG, dan CH 4 diagonal ruang yaitu AG, HB, DF dan CE Panjang diagonal sisi = s√2  Panjang diagonal ruang = s√3  

Pembahasan

Kubus ABCD.EFGH dengan panjang rusuk 4 cm, memiliki

Panjang diagonal sisi = 4√2 cm Panjang diagonal ruang = 4√3 cm

a) jarak titik A ke bidang CDGH

karena rusuk AD tegak lurus bidang CDHG, maka jarak titik A ke bidang CDGH adalah

= AD

= 4 cm

b) Jarak titik A ke bidang BCGF

karena rusuk AB tegak lurus bidang BCGF, maka jarak titik A ke bidang BCGF adalah

= AB  

= 4 cm  

c) jarak titik B ke bidang ACGE

misal O titik tengah AC dan titik tengah BD, maka BO tegak lurus bidang BCGF, sehingga jarak titik B ke bidang ACGF adalah

= BO

= ½ BD

= ½ (4√2 cm)

= 2√2 cm

Atau Cara Lain  

Pada segitiga ABC siku-siku di B

Jika alasnya AB, maka tingginya BC Jika alasnya AC, maka tingginya BO

dengan kesamaan luas, diperoleh nilai BO yaitu

½ × AC × BO = ½ × AB × BC

BO = [tex]\frac{AB \times BC}{AC}[/tex]

BO = [tex]\frac{4 \times 4}{4 \sqrt{2}} \: cm[/tex]

BO = [tex]\frac{4}{\sqrt{2}} \: cm[/tex]

BO = [tex]\frac{4}{\sqrt{2}} \times \frac{\sqrt{2}}{\sqrt{2}} \: cm[/tex]

BO = [tex]\frac{4 \sqrt{2}}{2} \: cm[/tex]

BO = 2√2 cm

d. jarak titik C ke bidang BDG

Perhatikan segitiga siku-siku GOC, dengan ukuran

GC = 4 cm OC = ½ AC = ½ (4√2 cm) = 2√2 cm

GO = [tex]\sqrt{GC^{2} + OC^{2}}[/tex]

GO = [tex]\sqrt{4^{2} + (2 \sqrt{2})^{2}} \: \: cm[/tex]

GO = [tex]\sqrt{16 + 8} \: \: cm[/tex]

GO = [tex]\sqrt{24} \: \: cm[/tex]

GO = [tex]\sqrt{4 \times 6} \: \: cm[/tex]

GO = [tex]2 \sqrt{6} \: \: cm[/tex]

Pada segitiga tersebut

Jika alasnya OC maka tingginya GC Jika alasnya GO, maka tingginya t (t adalah jarak titik C ke bidang BDG)

Dengan kesamaan luas segitiga GOC, maka diperoleh nilai c yaitu

½ × GO × t = ½ × OC × GC

t = [tex]\frac{OC \times GC}{GO}[/tex]

t = [tex]\frac{2 \sqrt{2} \times 4}{2 \sqrt{6}} \: cm[/tex]

t = [tex]\frac{4}{\sqrt{3}} \: cm[/tex]

t = [tex]\frac{4}{\sqrt{3}} \times \frac{\sqrt{3}}{\sqrt{3}} \: cm[/tex]

t = [tex]\frac{4}{3}\sqrt{3} \: cm[/tex]

Jadi jarak titik C ke bidang BDG adalah [tex]\frac{4}{3}\sqrt{3} \: cm[/tex]

e. Jarak titik E ke BDG

Perhatikan segitiga EOG, dengan ukuran  

EG = 4√2 cm EO = GO = 2√6 cm (sudah di cari di option d)

Misal P titik tengah EG, maka pada segitiga EOG

Jika alasnya EG, maka tingginya PO (PO = panjang rusuk kubus = 4 cm) Jika alasnya GO, maka tingginya t (t adalah jarak titik E ke bidang BDG)

Dengan kesamaan luas segitiga, maka diperoleh nilai t yaitu

½ × GO × t = ½ × EG × PO

t = [tex]\frac{EG \times PO}{GO}[/tex]

t = [tex]\frac{4 \sqrt{2} \times 4}{2 \sqrt{6}} \: cm[/tex]

t = [tex]\frac{8}{\sqrt{3}} \: cm[/tex]

t = [tex]\frac{8}{\sqrt{3}} \times \frac{\sqrt{3}}{\sqrt{3}} \: cm[/tex]

t = [tex]\frac{8}{3}\sqrt{3} \: cm[/tex]

Jadi jarak titik E ke bidang BDG adalah [tex]\frac{8}{3}\sqrt{3} \: cm[/tex]

f. jarak titik F ke bidang BDG

Buat kubus dengan sisi alas FGCB, O titik tengah BG Perpanjang garis DO dan garis EF sehingga perpanjangannya berpotongan di titik E’

Perhatikan segitiga FOE’

Jika alasnya FO maka tingginya FE’ Jika alasnya OE’, maka tingginya t (t adalah jarak titik F ke bidang BDG)

Dengan kesamaan luas segitiga GOC, maka diperoleh nilai t yaitu

½ × OE’ × t = ½ × FO × FE’

t = [tex]\frac{FO \times FE'}{OE'}[/tex]

t = [tex]\frac{2 \sqrt{2} \times 4}{2 \sqrt{6}} \: cm[/tex]

t = [tex]\frac{4}{\sqrt{3}} \: cm[/tex]

t = [tex]\frac{4}{\sqrt{3}} \times \frac{\sqrt{3}}{\sqrt{3}} \: cm[/tex]

t = [tex]\frac{4}{3}\sqrt{3} \: cm[/tex]

Jadi jarak titik F ke bidang BDG adalah [tex]\frac{4}{3}\sqrt{3} \: cm[/tex]

 

g. jarak titik H ke bidang BDG (cara sama dengan bagian f)

Buat kubus dengan sisi alas HDCG, O titik tengah GD Perpanjang garis BO dan garis EH sehingga perpanjangannya berpotongan di titik E’

Perhatikan segitiga HOE’

Jika alasnya HO maka tingginya HE’ Jika alasnya OE’, maka tingginya t (t adalah jarak titik H ke bidang BDG)

Dengan kesamaan luas segitiga GOC, maka diperoleh nilai t yaitu

½ × OE’ × t = ½ × HO × HE’

t = [tex]\frac{HO \times HE'}{OE'}[/tex]

t = [tex]\frac{2 \sqrt{2} \times 4}{2 \sqrt{6}} \: cm[/tex]

t = [tex]\frac{4}{\sqrt{3}} \: cm[/tex]

t = [tex]\frac{4}{\sqrt{3}} \times \frac{\sqrt{3}}{\sqrt{3}} \: cm[/tex]

t = [tex]\frac{4}{3}\sqrt{3} \: cm[/tex]

Jadi jarak titik H ke bidang BDG adalah [tex]\frac{4}{3}\sqrt{3} \: cm[/tex]

Cara cepat

Untuk jawaban bagian d, f dan g

Jarak titik C ke bidang BDG

= Jarak titik H ke bidang BDG

8. Tolong dengan pembahasannya juga kak, danke yak


Diketahui limas persegi T.PQRS dengan panjang rusuk alas: PQ = QR = RS = PR= 4 cm dan panjang rusuk tegak: TP = TQ = TR = TS = 6 cm. Jarak dari  

a. titik T ke garis PR = [tex]2\sqrt{7} \: \: cm[/tex] b. titik T ke garis PQ = [tex]4\sqrt{2} \: \: cm[/tex] c. titik P ke garis TQ = [tex]\frac{8}{3} \sqrt{2} [/tex] cm d. titik P ke garis TR = [tex]\frac{4}{3} \sqrt{14} [/tex] cm e. titik P ke garis OT = [tex]2\sqrt{2} \: \: cm[/tex] f. titik P ke garis QS = [tex]2\sqrt{2} \: \: cm[/tex]

Pembahasan

a. Jarak titik T ke garis PR

PR = SQ, diperoleh dengan menggunakan rumus Pythagoras yaitu

PR =  [tex]\sqrt{PQ^{2} + QR^{2}}[/tex]

PR =  [tex]\sqrt{4^{2} + 4^{2}} \: \: cm[/tex]

PR =  [tex]\sqrt{16 + 16} \: \: cm[/tex]

PR =  [tex]\sqrt{32} \: \: cm[/tex]

PR =  [tex]\sqrt{16 \times 2} \: \: cm[/tex]

PR =  [tex]4\sqrt{2} \: \: cm[/tex]

O titik tengah PR, maka

OP = OR = ½ PR = ½ (4√2 cm) = 2√2 cm

Jarak titik T ke garis PR adalah

TO = [tex]\sqrt{TR^{2} - OR^{2}}[/tex]

TO =  [tex]\sqrt{6^{2} - (2 \sqrt{2})^{2}} \: \: cm[/tex]

TO =  [tex]\sqrt{36 - 8} \: \: cm[/tex]

TO =  [tex]\sqrt{28} \: \: cm[/tex]

TO =  [tex]\sqrt{4 \times 7} \: \: cm[/tex]

TO =  [tex]2\sqrt{7} \: \: cm[/tex]

b. Jarak titik T ke garis PQ

Misal X titik tengah garis PQ, maka

XP = XQ = ½ PQ = ½ (4 cm) = 2 cm

Jarak titik T ke garis PQ adalah

TX = [tex]\sqrt{TQ^{2} - XQ^{2}}[/tex]

TX =  [tex]\sqrt{6^{2} - 2^{2}} \: \: cm[/tex]

TX =  [tex]\sqrt{36 - 4} \: \: cm[/tex]

TX =  [tex]\sqrt{32} \: \: cm[/tex]

TX =  [tex]\sqrt{16 \times 2} \: \: cm[/tex]

TX =  [tex]4\sqrt{2} \: \: cm[/tex]

c. Jarak titik P ke garis TQ

Misal jarak titik P ke garis TQ = t

Perhatikan segitiga PTQ dengan X adalah titik tengah PQ

Jika alasnya PQ, maka tingginya TX Jika alasnya TQ, maka tingginya t

Dengan kesamaan luas segitiga diperoleh rumus berikut:

½ × TQ × t = ½ × PQ × TX

t = [tex]\frac{PQ \: \times \: TX}{TQ}[/tex]

t = [tex]\frac{4 \: \times \: 4 \sqrt{2}}{6}[/tex] cm

t = [tex]\frac{16 \sqrt{2}}{6}[/tex] cm

t = [tex]\frac{8}{3} \sqrt{2} [/tex] cm

d. Jarak titik P ke garis TR

Misal jarak titik P ke garis TR = t

Perhatikan segitiga PTR  

Jika alasnya PR, maka tingginya TO Jika alasnya TR, maka tingginya t

Dengan kesamaan luas segitiga diperoleh rumus berikut:

½ × TR × t = ½ × PR × TO

t = [tex]\frac{PR \: \times \: TO}{TR}[/tex]

t = [tex]\frac{4 \sqrt{2} \: \times \: 2 \sqrt{7}}{6}[/tex] cm

t = [tex]\frac{8 \sqrt{14}}{6}[/tex] cm

t = [tex]\frac{4}{3} \sqrt{14} [/tex] cm

e. Jarak titik P ke garis OT

karena garis PO tegak lurus garis OT, maka jarak titik P ke garis OT adalah

= PO

= ½ PR

= ½ (4√2 cm)  

= 2√2 cm

f. Jarak titik P ke garis QS

karena garis PO tegak lurus garis QS, maka jarak titik P ke garis QS adalah

= PO

= ½ PR

= ½ (4√2 cm)  

= 2√2 cm

Diketahui kubus ABCD.EFGH dengan panjang rusuk = 4 cm. Jarak dari titik A ke  

a. garis HB = [tex]\frac{4}{3}\sqrt{6} [/tex] cmb. garis BD = [tex]2 \sqrt{2} [/tex] cmc. garis CH = [tex]2\sqrt{6} \: \: cm[/tex]d. garis CG = [tex]4 \sqrt{2} [/tex] cme. garis GH = [tex]4 \sqrt{2} [/tex] cm

Pada kubus ABCD.EFGH dengan panjang rusuk = s cm, maka

Panjang diagonal sisi (ds) = s√2 cmPanjang diagonal ruang (dr) = s√3 cm

Pembahasan

a. Jarak titik A ke garis HB

buat segitiga AHB dan segitiga tersebut siku-siku di titik A, dengan ukuran

AB = 4 cmAH = 4√2 cm ⇒ (diagonal sisi)HB = 4√3 cm ⇒ (diagonal ruang)

21. apakah kegunaan dari perintah dir


Kegunaan Perintah Dir Pada DOS
Saat ini perintah Dir pada DOS masih digunakan meskipun terdapat fasilitas GUI pada PC. Berikut ini perintah-perintah dir pada windows, antara lain :
1. dir c:
Ini adalah perintah dasar yang digunakan untuk menampilkan file dan directory pada drive C.
2. dir c: d:
Digunakan untuk menampilkan file dan directory pada drive C dan D.
3. dir /b /s
Digunakan untuk menampilkan directory dan subdirectory dengan tampilan full path name.
4. dir /a
Digunakan untuk menampilkan semua file termasuk file system dan file yang tersembunyi (hidden).
Untuk menampilkan semua file yang ada di drive D gunakan perintah sbb: dir d: /a
5. dir /os
Digunakan untuk menampilkan file dan directory dengan urutan mulai dari file yang berukuran paling kecil sampai dengan file yang berukuran paling besar. Untuk membalik urutan Anda bisa menggunakan perintah: dir /o-s
Anda bisa mengganti parameter s pada perintah di atas dengan:
- n, perintahnya dir /on : urut nama
- e, perintahnya dir /oe: urut tipe file
- d, perintahnya dir /od: urut tanggal
6. dir /p
Untuk menampilkan file dan directory per “halaman”.

Mata Pelajaran: Teknologi Informasi
Kata kunci: Perintah DOS

• Jawaban pendek:

Pada sistem operasi untuk menampilkan susunan file dapat diurutkan berdasarkan extensi file, melalui perintah DOS untuk menampilkan susunan file diurutkan berdasar extensi adalah ….

A. DIR /O E

• Penjelasan: 

Perintah DIR
Jika digunakan tanpa parameter, perintah ini dapat menampilkan daftar berkas-berkas dan subdirektori yang terdapat di dalam direktori aktif. Berkas ini memiliki satu parameter, yakni lokasi direktori di mana hendak menampilkan daftar isi direktori.

Jika ingin menampilkan susunan file berdasarkan ekstensi, perintahnya adalah:

dir /o e
