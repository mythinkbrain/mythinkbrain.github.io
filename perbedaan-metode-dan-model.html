4. Sebuah produsen sepatu membuat 2 model sepatu menggunakan 2 bahan yang berbeda. Komposisi model yang pertama terdiri dari 200 gr bahan pertama dan bahan kedua 150 gr. Sedangkan komposisi model kedua tersebut terdiri dari 180 gr bahan pertama dan 170 gr bahan kedua. Persediaan di gudang bahan pertama 76 kg dan persediaan digudang untuk bahan kedua 64 kg. Keuntungan model pertama ialah Rp. 50.000,00 dan untuk model kedua harganya Rp. 40.000,00.Tentukan x dan y yang memaksimalkan keuntungan dan berapa keuntungan maksimalnya dengan menggunakan metode grafik dan metode simpleks.​


Untuk mendapatkan keuntungan maksimal, maka x = 0,38 dan y = 0

Penjelasan dengan langkah-langkah:

Model Pertama (x) :

Bahan Pertama = 200 gr

Bahan Kedua = 150 gr

Model Kedua (y) :

Bahan Pertama = 180 gr

Bahan Kedua = 170 gr

Persamaan :

200x + 180y ≤ 76150x + 170y ≤ 64f(x,y) = 50.000x + 40.000y

Mencari Titik Potong

200x + 180y = 76

x = 0 , maka y = 0,42 ⇒ (0;0,42)

y = 0 , maka x = 0,38 ⇒ (0,38;0)

150x + 170y = 64

x = 0 , maka y = 0,38 ⇒ (0;0,38)

y = 0 , maka x = 0,43 ⇒ (0,43;0)

Eliminasi kedua persamaan diatas untuk menentukan titik potong (x,y)

200x + 180y = 76  (x3)       600x + 540y = 228

150x  + 170y = 64  (x4)       600 x + 680y= 256 -

                                                       -140y = -28   ⇒ y = 0,2

150x + 180 (0,2) = 76 ⇒ x ≈ 0,2 (0,2 ; 0,2)  

Menggambar Grafik (Lampiran)

Uji titik - lalu memilih daerah dengan arsiran terbanyak

(0,0) ⇒ Untuk persamaan 1 → 0 ≤ 76 (benar)

                     Untuk persamaan 2 ⇒ 0 ≤ 64 (benar)

Telah didapatkan titik titik yang akan disubstitusikan dalam persamaan keuntungan, dan di cari keuntungan terbesarnya - titik titik tersebut adalah batas dari daerah dengan arsiran terbanyak

Mencari keuntungan maksimal

f(x,y) = 50.000x + 40.000y

f(0;0,38) = 0 + 40.000 x 0,38 = 15.200

f(0,2;0,2) = 50.000 x 0,2 + 40.000 x 0,2 = 18.000

f(0,38;0) = 50.000 x 0,38 + 0 = 19.000

Maka, didapatkan x = 0,38 dan y = 0 untuk memperoleh keuntungan maksimal

Jawaban:

Model waterfall adalah pendekatan linier untuk pengembangan perangkat lunak di mana setiap tahap proses diselesaikan sebelum pindah ke tahap berikutnya. Ini berarti bahwa semua persyaratan harus dikumpulkan di awal, dan setiap perubahan yang dibuat selama pengembangan harus dimulai dari awal. Keuntungan dari pendekatan ini adalah memungkinkan pencapaian dan pencapaian yang jelas, membuatnya mudah untuk melacak kemajuan dan mengidentifikasi potensi masalah sejak dini. Namun, kelemahannya adalah tidak fleksibel dan tidak memungkinkan banyak iterasi atau perubahan setelah pengembangan dimulai.

Model inkremental adalah pendekatan yang lebih fleksibel di mana pekerjaan dibagi menjadi peningkatan kecil atau fase, dengan setiap fase dibangun di atas yang sebelumnya. Ini berarti bahwa fitur baru dapat ditambahkan pada setiap peningkatan, dan umpan balik dari pengguna dapat dimasukkan ke dalam fase pengembangan di masa mendatang. Keuntungan dari pendekatan ini adalah memungkinkan lebih banyak fleksibilitas dan masukan pengguna selama pengembangan; namun, sisi negatifnya adalah dapat memakan waktu lebih lama secara keseluruhan untuk menyelesaikan sebuah proyek karena setiap fase harus diselesaikan sepenuhnya sebelum pindah ke yang berikutnya.

Model spiral adalah proses pengembangan perangkat lunak yang menggabungkan elemen desain dan pembuatan prototipe secara bertahap, dalam upaya untuk menggabungkan keunggulan konsep top-down dan bottom-up. Model spiral mirip dengan model inkremental, dengan lebih menekankan pada analisis risiko. Model spiral memiliki empat fase: Perencanaan, Analisis Risiko, Teknik dan Evaluasi.

Model pengembangan aplikasi cepat (RAD) adalah jenis metodologi pengembangan perangkat lunak tangkas yang berfokus pada pengiriman cepat produk perangkat lunak berkualitas tinggi. Pendekatan RAD mencakup pengembangan berulang dan inkremental, serta keterlibatan pengguna yang ekstensif sepanjang siklus hidup proyek.
