11. [+50][tex]\lim _ { x \rightarrow 0 } \frac{ \sin(6x) }{ \sin(2x) } [/tex]gunakan aturan L'hospital​


limit

• L'Hopital

lim x→0 sin 6x / sin 2x

= lim x→0 6 cos 6x / 2 cos 2x

= 3 cos (6.0) / cos (2.0)

= 3 cos 0 / cos 0

= 3

Matematika

______________________

Penyelesaian

Menyelesaiakan pertanyaan limit tersebut menggunakan aturan L'hospital :

= Lim x-->0 sin 6x/ sin 2x

= Lim x-->0 6 cos 6x/ 2 cos 2x

= 3 cos (6 × 0) / cos (2 × 0)

= 3 cos 0 / cos 0

= 3

Jadi, nilainya adalah 3
