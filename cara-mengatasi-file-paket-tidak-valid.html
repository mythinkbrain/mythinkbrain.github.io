7. excel cannot open. because the file format or file extension is not valid. verify that the file has not been corrupted and that the file extension matches the format of the file. Bagaimana caranya?​tolong di jawab karna ini tugas penting


Jawaban:

Format ekstensi dari file yang ingin kamu buka tidak di support oleh excel.

Ms. Excel support format .xls .xlsx .xlsm .xlmb .xltx .xltm .xlt .xml dsb

Ini bisa dilihat di akhir nama file, contoh :

"tugas penting.xlsx" maka file tersebut bisa dibuka oleh Ms. Excel.

"tugas penting.exe" tidak bisa dibuka oleh Ms. Excel karena format ekstensi filenya tidak disupport.

Selain itu, bisa juga file yang ingin kamu buka sudah corrupted atau rusak ketika disalin atau karena virus. Bisa coba download atau salin ulang tugasnya.

13. Cara mengatasi thumbnail file hilang di windows 10 gimana?​


Jawaban:

CARA 1

Sebelum menggunakan cara yang lebih kompleks, ada baiknya kita mengecek terlebih dahulu apakah dukungan  gambar mini diaktifkan oleh File Explorer. Namun, terkadang tiap folder memiliki pengaturan tersendiri sehingga jika Anda mengatur secara global, pada folder tertentu tetap tidak menampilkan thumbnail.

Untuk mengaktifkan gambar cuplikan, klik tab View lalu klik salah satu pilihan di bawah ini:

Extra large icons Large icons Medium icons Tiles Content[ LIHAT GAMBAR 1 ]

Semua mode tampilan di atas mendukung gambar cuplikan. Pilih salah satu diantara pilihan di atas untuk mengaktifkan gambar mini.

CARA 2

Meskipun sudah menggunakan mode tampilan yang mendukung gambar mini, thumbnail sering kali tetap tidak diaktifkan karena File Explorer dikonfigurasi agar hanya menampilkan icon saja tidak dengan cuplikan. Ketika pengaturan tersebut tidak diaktifkan, File Explorer hanya menampilkan icon dari aplikasi yang digunakan untuk membuka file yang berkaitan.

Berikut adalah cara mengaktifkan thumbnail di Windows 10

Langkah 1: Buka File Explorer. Klik menu File lalu klik Change folder and search options untuk membuka pengaturan folder. Ketika ada file yang sedang terpilih, Windows 10 hanya akan menampilkan Options saja bukan “Change folder and search options”. Klik menu tersebut juga akan membuka jendela yang sama.

[ LIHAT GAMBAR 2 ]

Langkah 2: Pindah ke tab View. Dibagian Advanced settings, hilangkan centang pada opsi dengan judul Always show icons, Never thumbnails, dan klik tombol Apply untuk menerapkan perubahan.

[ LIHAT GAMBAR 3 ]

CARA 3

Jika tampilan gambar mini terlihat tidak jelas (blur), kemungkinan besar hal tersebut dikarenakan file cache thumbnail yang rusak. Anda dapat membersihkan dan mereset cache thumbnail melalui Disk Cleanup dengan mencentang pilihan Thumbnails.

CARA 4

Mungkin Anda sudah mengetahui, File Explorer di Windows 10 tidak dapat menampilkan gambar mini untuk semua jenis file. Meskipun sebagian besar tipe file gambar / video sudah di dukung, format seperti .mkv dan .flv sering kali ditampilkan hanya icon-nya saja.

Jika Anda menginginkan gambar mini untuk semua jenis file gambar dan video, maka Anda harus menginstall thumbnail generator yang dijuluki Icaros. Aplikasi mungil tersebut tersedia secara gratis dan mampu membuat thumbnail hingga 15 jenis file.

Penjelasan:

Sumber : https://www.aplikasipc.com/mengatasi-gambar-mini-thumbnail-tidak-muncul-di-file-explorer/


14. Cara Mengatasi File Photo Yang Tidak bisa di buka"The app didn't start"?


'The app didn't start' adalah salah satu pesan kesalahan yang ditampilkan oleh sebuah perangkat lunak yang menandakan bahwa aplikasi tersebut tidak dapat dijalankan. Peristiwa ini dapat terjadi karena beberapa hal, mulai dari kehilangan bagian (berkas .dll atau yang lainnya) maupun kekurangan memori yang dibutuhkan untuk menjalankan aplikasi.

Pembahasan

Pada kesempatan ini, soal meminta kita untuk menyajikan cara mengatasi file photo yang tidak bisa dibuka dengan menampilkan pesan 'the app didn't start'. Berikut kakak akan mencoba menjawab pertanyaan tersebut.

CARA MENGATASI PESAN KESALAHAN 'THE APP DIDN'T START' JIKA TERJADI KEKURANGAN MEMORI:

MENGKLIK OK UNTUK MENUTUP PESAN KESALAHAN YANG MUNCUL. TUTUP APLIKASI LAIN YANG SEDANG BERJALAN DAN MENGONSUMSI MEMORI DALAM JUMLAH BESAR SEHINGGA MENYISAKAN RUANG YANG CUKUP UNTUK BERJALANNYA APLIKASI PEMBUKA FOTO.BUKA KEMBALI BERKAS FOTO/

CARA MENGATASI PESAN KESALAHAN 'THE APP DIDN'T START' JIKA TERJADI KERUSAKAN APLIKASI:

TUTUP PESAN KESALAHAN YANG DIMUNCULKAN APLIKASIHAPUS DAN PASANG ULANG APLIKASI UNTUK MEMASTIKAN BAHWA SEMUA BAGIAN APLIKASI TELAH TERPASANG DENGAN BAIK DAN TIDAK MENGALAMI KESALAHANBUKA KEMBALI BERKAS FOTO DENGAN APLIKASI YANG SAMA.

19. Bagaimana cara mengatasi file exe yang tidak ada di visual studio ?


Penjelasan:

:

1. Buka project yang sudah dibuat

2. Pilih menu Project – (klik nama project) Properties

3. Setelah jendela properties dari project Anda terbuka kemudian pilih menu publish dari jendela tersebut

4. tentukan dimana lokasi file hasil publish disimpan pada “Publish Location”

5. Pilih Publish Now (tunggu hingga Visual Studio selesai mempublish project Anda menjadi file setup.exe.)

6. Cara bukanya di (nama project anda)/bin/debug/app.publish

7. Setelah mempunyai file setup dari aplikasi yang di buat, selanjutnya install di komputer yang diinginkan dengan mengklik file setup yang ada.

30. cara mengatasi file ppt yg corrupt


Buka program ppt, buka file ppt baru (file-new), pada tab home, klik lambang ujung anak panah dibagian bawah menu new slide pilih reuse slides. Pada panel dibagian bawah layar klik browse untuk membuka file ppt yg corrupt tadi. Setelah file tadi ter-load, kemudian klik kanan pada salah satu slide yang muncul, pilih insert all slides, kemudian simpan file dengan nama yang baru. Kini anda telah berhasil memindahkan isi dari file yang corrupt ke file yang baru.
