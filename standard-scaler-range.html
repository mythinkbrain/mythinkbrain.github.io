6. adalah 4,3%; 4,9%; 7,2%; 6,7%; dan 11,6%. Hitunglah range, mean deviation, standard deviation, dan coefficient of variation.​


Jawab:

Range: 7,3%

Mean deviation: -0,74%

Standard deviation: 1,57%

Coefficient of variation: 22%

Penjelasan dengan langkah-langkah:

Range: Range adalah selisih antara nilai terbesar dan terkecil dari data. Dalam kasus ini,

range = 11,6% - 4,3%

          = 7,3%.

jadi, range : 7,3%

Mean deviation:

Nilai rata-rata dari data tersebut adalah

(4,3% + 4,9% + 7,2% + 6,7% + 11,6%) / 5

= 7,14%

Selisih antara setiap nilai dengan nilai rata-rata adalah:

4,3% - 7,14% = -2,84%

4,9% - 7,14% = -2,24%

7,2% - 7,14% = 0,06%

6,7% - 7,14% = -0,44%

11,6% - 7,14% = 4,46%

Mean deviation

(-2,84% + -2,24% + 0,06% + -0,44% + 4,46%) / 5

= -0,74%.

jadi, Mean deviation: -0,74%

Standard deviation

Rumus untuk menghitung standard deviation adalah:

SD = √(((x1 - x̄)^2 + (x2 - x̄)^2 + ... + (xn - x̄)^2) / (n - 1))

dimana:

x1, x2, ..., xn = nilai-nilai dari data

x̄ = nilai rata-rata dari data

n = jumlah data

Jadi, standard deviation dari data tersebut adalah:

SD = √(((-2,84% - (-0,74%))^2 + (-2,24% - (-0,74%))^2 + (0,06% - (-0,74%))^2 + (-0,44% - (-0,74%))^2 + (4,46% - (-0,74%))^2) / (5 - 1))

= √((3,1^2 + 1,5^2 + 0,8^1 + 0,3^2 + 5,2^2) / 4)

= √(9,61 + 2,25 + 0,64 + 0,09 + 27,04) / 4

= √(39,54) / 4

= 6,27% / 4

= 1,57%

jadi, Standard deviation: 1,57%

Coefficient of variation

standard deviation/Nilai rata-rata dari data

1,57% / 7,14%  

= 0,22 atau 22%.

jadi, Coefficient of variation: 22%

18. mengapa interface setiap aplikasi wajib mengikuti standard? apabila perancang interface tidak mengikuti standard , apa yang akan terjadi?


Mengapa interface setiap aplikasi wajib mengikuti standard? apabila perancang interface tidak mengikuti standard , apa yang akan terjadi?

Pembahasan

Jawabannya adalah mengapa harus mengikuti standard karena standard interface setiap aplikasi sudah mengalami riset, dan uji coba yang dilakukan kepada user. Sehingga telah diketahui bahwa standar tersebut adalah yang telah disesuaikan dengan kebutuhan dan kenyamanan, serta keuntungan lain bagi bagi para developer yaitu tidak perlu mengikuti riset kembali karena standar tersebt telah diciptakan.

Apa bila developer atau perancang interface tidak mengikuti standard maka resiko yang akan terjadi adalah menambah banyak kemungkinan apabil interface tersebut tidak diterima oleh para pengguna, dan mengakibatkan user kesuitan menggunakan interface tersebut. Resiko lain juga membuat developer harus melakukan riset yang notabennya membutuhkan waktu yang tidak sebentar.

30. Sebuah perusahaan membagikan laba keuntungan perusahaan dalam 5 tahun terakhir adalah 4,3%; 4,9%; 7,2%; 6,7%; dan 11,6%. Hitunglah range, mean deviation, standard deviation, dan coefficient of variation.​


Jawab:

Range: 7,3%

Mean deviation: -0,74%

Standard deviation: 1,57%

Coefficient of variation: 22%

Penjelasan dengan langkah-langkah:

Range : nilai terbesar - nilai terkecil

11,6% - 4,3%

= 7,3%.

Mean deviation: Mean deviation adalah rata-rata dari selisih antara setiap nilai dengan nilai rata-rata

Nilai rata-rata

=(4,3% + 4,9% + 7,2% + 6,7% + 11,6%) / 5

= 7,14%

Selisih antara setiap nilai dengan nilai rata-rata adalah:

4,3% - 7,14% = -2,84%

4,9% - 7,14% = -2,24%

7,2% - 7,14% = 0,06%

6,7% - 7,14% = -0,44%

11,6% - 7,14% = 4,46%

Mean deviation

=(-2,84% + -2,24% + 0,06% + -0,44% + 4,46%) / 5

= -0,74%.

Standar deviation

Rumus untuk menghitung standard deviation adalah:

SD = √(((x1 - x̄)^2 + (x2 - x̄)^2 + ... + (xn - x̄)^2) / (n - 1))

dimana:

x1, x2, ..., xn adalah nilai-nilai dari data

x̄ adalah nilai rata-rata dari data

n adalah jumlah data

Jadi, standard deviation dari data tersebut adalah:

SD = √(((-2,84% - (-0,74%))^2 + (-2,24% - (-0,74%))^2 + (0,06% - (-0,74%))^2 + (-0,44% - (-0,74%))^2 + (4,46% - (-0,74%))^2) / (5 - 1))

= √((3,1^2 + 1,5^2 + 0,8^1 + 0,3^2 + 5,2^2) / 4)

= √(9,61 + 2,25 + 0,64 + 0,09 + 27,04) / 4

= √(39,54) / 4

= 6,27% / 4

= 1,57%

Coefficient of variation :

1,57% / 7,14%

= 0,22 atau 22%.
