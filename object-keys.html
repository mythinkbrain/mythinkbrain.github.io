12. apa yang dimaksud dengan Home keys


Yang dimaksud dengan home keys adalah tombol-tombol pada keyboard komputer atau laptop yang letaknya merupakan posisi jari-jari pengguna pada saat dalam posisi diam. Contoh home keys untuk keyboard dengan layout QWERTY adalah tombol huruf A, S, D, dan F untuk posisi tangan kiri dan tombol huruf J, K, L, dan ; (semicolon) untuk posisi tangan kanan. Sebagai patokan jari kita, pada tombol keyboard huruf F dan J biasanya terdapat tonjolan untuk membantu meletakkan jari kita tanpa perlu melihat keyboard.

Kelompok tombol pada keyboard

Selain home keys, dua bagain lain dari layout keyboard adalah:

1. Top row keys

adalah tombol-tombol keyboard yang terletak di atas home keys. Top row keys inilah yang menentukan jenis layput dari suatu keyboard apakah QWERTY, AZERTY atau QWERTZ. Dalam layout QWERTY, top row keys meliputi tombol huruf Q, W, E, R, T, Y, U, I, O, P.

2. Bottom row keys

adalah tombol-tombol keyboard yang posisinya berada di bawah tombol home keys. Pada keyboard dengan layout QWERTY, tombol-tombol terebut adalah A, X, C, V, B, N M, koma, titk, dan slash.

Halo guys, seperti yang anda ketahui, di saat ini banyak aneka macam software yang berguna untuk menunjang kegiatan desain. salah satunya yaitu CorelDraw. CorelDraw adalah software yang dipergunakan untuk membuat gambar atau desain menggunakan basis vector. aplikasi ini mempunyai banyak kegunaan dalam urusan grafis seperti, membentuk sketsa, gambar, logo, sampai desain arsitektur.

CorelDraw merupakan editor grafik vektor yang dikembangkan oleh Corel, sebuah perusahaan software yang berkantor pada Ottawa, Kanada. Versi terbarunya, CorelDRAW X8 (setara dengan versi 18) dirilis pada tanggal 15 Maret 2016. CorelDRAW pada awalnya dikembangkan untuk sistem operasi Windows 2000 serta seterusnya.

Versi CorelDRAW untuk Mac OS pernah dikembangkan, namun dilarang karena taraf penjualannya rendah. Versi terakhir untuk Linux ialah versi 9 (Rilisan tahun 2000, itu pun perlu di jalankan dengan Wine) serta untuk OSX adalah versi 11 (Rilisan 2001)
