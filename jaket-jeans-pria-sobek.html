30. Harga satu jeans dan dua kaos Rp150.000,- sedangkan harga 3 jeans dan 3 kaos Rp270.000,- . Harga 2 jeans dan 5 jeans adalah.


Jawab:

Jeans = x

Kaos = y

Diketahui :  x + 2y = 150.000

                 3x + 3y = 270.000

Ditanya :  2x + 5y

Dijawab :

x + 2y  = 150.000  ║ × 3

3x + 3y = 270.000  ║ × 1

-----------------------------------

3x + 6y = 450.000

3x + 3y = 270.000

----------------------------- -

       3y = 180.000

         y = Rp 60.000  ( Harga 1 kaos = Rp 60.000 )

Ganti y ke salah satu persamaan

x + 2y = 150.000

x + 2(60.000) = 150.000

x + 120.000 = 150.000

x = 150.000 - 120.000

x = 30.000 ( Harga 1 jeans = Rp. 30.000 )

Karena soal kurang tepat maka saya ksih 2 soal

a) Harga 2 jeans dan 5 kaos (2x + 5y)b) Harga 2 kaos dan 5 jeans  (2y + 5x)

jawab :

a) Harga 2 jeans dan 5 kaos (2x + 5y)

⇔ 2(30.000) + 5(60.000)

⇔ 60.000 + 30.000

⇔ Rp 90.000

b) Harga 2 kaos dan 5 jeans  (2y + 5x)

⇔ 2(60.000) + 5(30.000)

⇔ 120.000 + 150.000

⇔ Rp 270.000
