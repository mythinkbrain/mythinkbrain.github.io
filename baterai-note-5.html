Arus Listrik

mengalirnya elektron secara terus menerus dan berkesinambungan pada konduktor akibat perbedaan jumlah elektron pada beberapa lokasi yang jumlah elektronnya tidak sama. satuan arus listrik adalah Ampere.

1 ampere arus adalah mengalirnya elektron sebanyak 1 Coulumb per detik melewati suatu penampang konduktor :

I = Q / t

dimana :

I   = arus listrik ( A )

Q = besarnya muatan listrik ( C )

t   = waktu ( s )

________________________________________________________

t = 24 jam = 24 x 3.600 s = 86.400 s

I = 5 A

ditanya :

Q

jawab :

Q  =  I . t

    =  5 x 86.400

Contoh :
