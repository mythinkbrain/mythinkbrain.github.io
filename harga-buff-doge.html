19. 1.Hitunglah luas bagian bangun datar yang diarsir dibawah inibamtu jamwab kakDoge-​


Luas bagian bangun datar yang diarsir dibawah ini adalah 42 cm².

Pendahuluan

Lingkaran adalah kumpulan semua titik-titik pada bidang datar yang berjarak sama terhadap suatu titik tertentu (titik pusat).

Rumus lingkaran

Keliling = 2 × π × r = π × d Luas = π × r²

Rumus persegi

Keliling = 4 × s Luas = s × s

Pembahasan

Diketahui :

Jari-jari 1/4 lingkaran (r) = 7 cm

Panjang sisi persegi (s) = 7 cm + 7 cm = 14 cm

Ditanya :

Luas yang diarsir ?

Penyelesaian :

Menentukan luas persegi

L persegi = s × s

                = 14 cm × 14 cm

                = 196 cm²

Menentukan luas lingkaran

Keempat ¹/₄ lingkaran digabungkan akan menjadi satu lingkaran utuh.

L lingkaran = π × r²

                  = [tex]\displaystyle\frac{22}{7}[/tex] × 7 × 7 cm²

                  = 22 × 7 cm²

                  = 154 cm²

Menentukan luas bagian bangun datar yang diarsir

L diarsir = L persegi - L lingkaran

             = 196 cm² - 154 cm²

             = 42 cm²

Jadi luas bagian bangun datar yang diarsir adalah 42 cm².

// Deklarasi variabel

float x, y;    // Koordinat posisi Baby Doge

float speedX;  // Kecepatan perpindahan horizontal

void setup() {

 size(400, 400);  // Ukuran jendela

 x = width / 2;   // Posisi awal Baby Doge di tengah horizontal

 y = height / 2;  // Posisi awal Baby Doge di tengah vertikal

 speedX = 2;      // Kecepatan perpindahan horizontal

}

void draw() {

 background(220);  // Warna latar belakang

 // Menggambar Baby Doge

 fill(255, 0, 0);       // Warna isi Baby Doge (merah)

 stroke(0);             // Warna garis pinggir Baby Doge (hitam)

 ellipse(x, y, 50, 50); // Menggambar lingkaran (tubuh Baby Doge)

 // Menggerakkan Baby Doge

 x += speedX;  // Mengubah posisi Baby Doge berdasarkan kecepatan horizontal

 // Membatasi pergerakan Baby Doge ketika mencapai batas layar

 if (x > width || x < 0) {

   speedX *= -1;  // Membalik arah pergerakan Baby Doge

 }

}
