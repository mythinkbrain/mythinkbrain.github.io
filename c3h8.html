2. Dalam 12,04 × 10²³ molekul C3H8. hitunglah :a. mol (n) C3H8b. massa (m) C3H8c. volume (v) C3H8 (STP)d. volume (v) C3H8 (RTP)e. jumlah atom H dalam C3H8*) Ar C : 12 H : 1​​


a. Jumlah partikel = mol x L (bil. Avogadro)

12,04 x 10^23 = mol x 6,02 x 10^23

mol = 12,04 x 10^23 / 6,02 x 10^23

= 2 mol

b. Massa = mol x Mr

Mr C3H8 = (3 x 12 g/mol) + (8 x 1 g/mol)

= 44 g/mol

Maka, massa = 2 mol x 44 g/mol

= 88 gram

c. Volume = tetapan gas ideal x mol

Tetapan pada STP = 22,4

Maka, volume = 22,4 x 2 mol

= 44,8 L

d. Tetapan pada RTP = 24

Maka, volume = 24 x 2 mol

= 48 L

e. Jumlah atom H pada C3H8

C3H8 ➞ 3C + 8H

Maka, terdapat 8 atom H

21. Pembakaran gas propana (C3H8) sesuai dengan persamaan reaksi berikut. C3H8 (g) + O2(g) → CO2(g) + H2O(g) Perbandingan volume C3H8 dan O2 adalah .


C3H8 (g) + O2(g) → CO2(g) + H2O(g)

perbandingan volume = perbandingan koefisien

maka, untuk mencari perbandingan volume, kita setarakan dahulu reaksi tersebut

langkah menyetarakan:

1. setarakan jumlah unsur selain O dan H, yaitu C

jumlah C di kiri = 3, sedangkan di kanan 1, berarti kita tambahkan koefisien 3 di kanan agar setara

C3H8 (g) + O2(g) → 3 CO2(g) + H2O(g)

2. setarakan jumlah unsur H

jumlah H di kiri = 8, sedangkan di kanan 2, berarti kita tambahkan koefisien 4 di kanan agar setara

secara matematis:

8 = 2x

8/2 = x

x = 4

C3H8 (g) + O2(g) → 3 CO2(g) + 4 H2O(g)

3. setarakan jumlah unsur O

jumlah O di kiri = 2, sedangkan di kanan 10, berarti kita tambahkan koefisien 5 di kiri agar setara

secara matematis:

2x = 10

x = 10/2

x = 5

C3H8 (g) + 5 O2(g) → 3 CO2(g) + 4 H2O(g)

maka, perbandingan volume C3H8 dan O2 adalah 1 : 5

25. Diketahui 4,4 gr gas C3H8, tentukan : a. Mr dari C3H8 b. Jumlah mol dari C3H8 c. Volume C3H8 pada keadaan standar/STP (25C, 1 atm.) Ar H = 1, Ar C = 12


Jawaban:

a. 44

b. 0,1 mol

c. 2,24 L

Penjelasan:

a. Mr C3H8 = ( 3 x Ar C ) + ( 8 x Ar H )

=> 36 + 8

=> 44

b. massa = n x Mr

4,4 = n x 44

=> 0,1 mol

c. V STP = n x 22,4

=> 0,1 x 22,4

=> 2,24 L

30. Pada pembakaran 4,4 gram propana (C3H8) dibebaskan kalor sebesar 2218 kJ, maka DHc0 C3H8 adalah


Jawaban:

mol propana = [tex]\frac{4,4}{44}[/tex] = 0,1 mol

ΔH°c = [tex]\frac{q}{n}[/tex]

ΔH°c = [tex]\frac{2218}{0,1}[/tex]

ΔH°c = 22.180 kJ/mol

C₃H₈ + 5O₂ → 3CO₂ + 4H₂O                   ΔH°c = 22.180 kJ/mol
