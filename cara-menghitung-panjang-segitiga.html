4. bagaimana cara menghitung panjang sisi segitiga pyhthagoras


Cara menghitung Phytagoras =
Rumus Pythagoras =
A. Sisi alas [a²] = c²-b²
B. Sisi atas [b²] = c²-a²
C. Sisi miring [c²] = a²+b²

Contoh soal : Sebuah segitiga siku-siku, sisi alasnya (a) 3cm dan sisi tegaknya (b) 4cm. Berapa sisi miringnya (c) ?

Jawab = Sisi Miring (gunakan rumus c²)
*C² = a²+b² = 3²+4² = 9+16 = 25
*C  = √25 (akar pangkat 2 dari 25) = 5

Jadi, C (Sisi miring) dari segitiga tersebut adalah 5

phytagoras untuk mencari ukuran sisi2 segitiga siku²

a=alas atau sisi datar
b= tinggi atau sisi tegak
c = sisi miring

c²=a²+b²  --> c =√a²+b²

b²=c²-a² ---> b=√c²-a²

a² =c²-b² ---> a=√c²-b²
