12. Sebutkan 4 tombol (size button) dan fungsinya


Kelas           : VIII (2 SMP)
Pelajaran     : TIK 
Kategori      : Identifikasi Tampilan Layar
Kata Kunci  : Size, Button, Minimize, Close, Restore



Terdapat 4 pilihan yang berkaitan dengan ukuran tampilan layar kerja pada Microsoft Word yang lazim disebut Size Button, yakni:

→ RESTORE 
→ MINIMIZE
→ MAXIMIZE
→ CLOSE

RESTORE, berfungsi dalam mengembalikan jendela kerja pada ukuran yang semula.

CLOSE, berfungsi menutup jendela dokumen kerja.

MINIMIZE, berfungsi memperkecil ukuran dari jendela kerja.

MAXIMIZE, berfungsi dalam memperbesar ukuran dari jendela kerja.

Keempat size button ini terdapat pada Tittle Bar atau baris judul. Size Button ini juga dikenal dengan istilah Kontrol Menu.

13. fungsi windows size button dalam Mozilla Firefox


Fungsi Windows Size Button dalam Mozilla Firefox untuk mengatur ukuran jendela browser Mozilla Firefox. Ada 3 button/tombol pada Windows Size Button:
- Close (tombol X): berfungsi untuk menutup jendela browser Mozilla Firefox 
- Maximize (tombol kotak)/Restore Down (tombol kotak ada 2): berfungsi untuk melebarkan tampilan jendela Mozilla Firefox (Maximize) dan untuk mengecilkan/mengembalikan tampilan jendela Mozilla Firefox ke ukuran sebelumnya/dari Maximize ke ukuran sebelum diMaximize (Restore Down) 
- Minimize (tombol - ): berfungsi untuk menyembunyikan jendela Mozilla Firefox ke latar belakang/background
