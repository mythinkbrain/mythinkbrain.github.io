3. Cara mengatasi thumbnail file hilang di windows 10 gimana?​


Jawaban:

CARA 1

Sebelum menggunakan cara yang lebih kompleks, ada baiknya kita mengecek terlebih dahulu apakah dukungan  gambar mini diaktifkan oleh File Explorer. Namun, terkadang tiap folder memiliki pengaturan tersendiri sehingga jika Anda mengatur secara global, pada folder tertentu tetap tidak menampilkan thumbnail.

Untuk mengaktifkan gambar cuplikan, klik tab View lalu klik salah satu pilihan di bawah ini:

Extra large icons Large icons Medium icons Tiles Content[ LIHAT GAMBAR 1 ]

Semua mode tampilan di atas mendukung gambar cuplikan. Pilih salah satu diantara pilihan di atas untuk mengaktifkan gambar mini.

CARA 2

Meskipun sudah menggunakan mode tampilan yang mendukung gambar mini, thumbnail sering kali tetap tidak diaktifkan karena File Explorer dikonfigurasi agar hanya menampilkan icon saja tidak dengan cuplikan. Ketika pengaturan tersebut tidak diaktifkan, File Explorer hanya menampilkan icon dari aplikasi yang digunakan untuk membuka file yang berkaitan.

Berikut adalah cara mengaktifkan thumbnail di Windows 10

Langkah 1: Buka File Explorer. Klik menu File lalu klik Change folder and search options untuk membuka pengaturan folder. Ketika ada file yang sedang terpilih, Windows 10 hanya akan menampilkan Options saja bukan “Change folder and search options”. Klik menu tersebut juga akan membuka jendela yang sama.

[ LIHAT GAMBAR 2 ]

Langkah 2: Pindah ke tab View. Dibagian Advanced settings, hilangkan centang pada opsi dengan judul Always show icons, Never thumbnails, dan klik tombol Apply untuk menerapkan perubahan.

[ LIHAT GAMBAR 3 ]

CARA 3

Jika tampilan gambar mini terlihat tidak jelas (blur), kemungkinan besar hal tersebut dikarenakan file cache thumbnail yang rusak. Anda dapat membersihkan dan mereset cache thumbnail melalui Disk Cleanup dengan mencentang pilihan Thumbnails.

CARA 4

Mungkin Anda sudah mengetahui, File Explorer di Windows 10 tidak dapat menampilkan gambar mini untuk semua jenis file. Meskipun sebagian besar tipe file gambar / video sudah di dukung, format seperti .mkv dan .flv sering kali ditampilkan hanya icon-nya saja.

Jika Anda menginginkan gambar mini untuk semua jenis file gambar dan video, maka Anda harus menginstall thumbnail generator yang dijuluki Icaros. Aplikasi mungil tersebut tersedia secara gratis dan mampu membuat thumbnail hingga 15 jenis file.

Penjelasan:

Sumber : https://www.aplikasipc.com/mengatasi-gambar-mini-thumbnail-tidak-muncul-di-file-explorer/

14. Cara mencari file file yang hilang pada laptop cara nya gimana?


Kemungkinan pertama. Kalau "hilang". Tinggal ketik nama file yang hilang di Search Bar di Windows Explorer (Yang ada tulisan "Search" di pojok kanan agak atas). Sebelum mengetik, pilih folder dimana kemungkinan file hilang. Kalau belum ketemu, ganti folder lain atau ganti ketikan nama yang dikira lebih sesuai.

Kemungkinan kedua. Ternyata bukan hilang, melainkan terhapus (belum permanen). Biasanya file masih terdapat di Recycle Bin di Windows Explorer. Tinggal cari gambar tempat sampah (Recycle Bin) >> Klik 2x >> Cari file yang terhapus >> Klik 2x (Perhatikan Origin, disitulah tempat file nanti kembali) >> Pilih Restore. File akan kembali ke lokasi di Origin tadi.

Kemungkinan ketiga. File terhapus permanen, sudah dicari di Recycle Bin tidak ada berarti file terhapus permanen dari komputer.

Kemungkinan tambahan. File terkena virus dan disembunyikan di suatu tempat. Ini agak susah, file yang disembunyikan virus besar kemungkinan kalau virusnya dihapus file ikut terhapus. Cari di google kalau yang ini.

25. Cara menginstal ulang laptop windows 7 tanpa CD room atau sebagainya dan tanpa menghilangkan data yang ada ?


Install ulang tanpa CD Room/Flashdisk bisa, tapi localdisk C otomatis terformat.
Carannya, download file .iso dari windows 7(32bit pake 32bit, 64bit pake 64bit, 32bit pakr 64bit atau sebaliknya gak bisa), taruh di localdisk D atau E.
Nanti akan muncul Window Install Windows
Tekan Install Now
Nanti layar Install Windows akan jadi full screen
Setelah itu akan ada pilihan mengenai Update, pilih saja yang Do Not ...
Setelah itu centang I Accept ....
Akan muncul pilihan lagi yakni Upgrade atau Custom(advance)
Pada pilihan ini anda disuruh untuk memilih option ingin Upgrade atau Custom. pilih Custom
Setelah itu muncul lagi Window untuk memilih Local Disc
Jika anda sudah sampai tahap ini, anda harus mencari Local Disk C: tempat system. Pastikan yang anda pilih adalah Local Disc C: . Jangan sampai salah. Unuk melihat mana C: , anda bisa melihat size dari Local Disc masing masing.
Baca : Nah disini mungkin letak perbedaan instal ulang dengan CD atau flashdisk. Jika anda pernah instal ulang menggunakan media tersebut, distep ini harusnya kita melakukan pemformatan pada Local Disc C:. Tapi disini berbeda, karna menginstal ulang menggunakan cara ini, kita membutuhkan file system mungin.
Setelah itu tekan next dan tinggal tunggu saja
Pada step ini anda tinggal menunggu saja prosesnya. Dan mungkin akan melakukan restart. Jika restart nanti akan disururuh Force Restart dan Cancel, pilih saja Force Restart. Setelah itu, akan tinggal diam diam dan menunggu proses selanjutnya.
