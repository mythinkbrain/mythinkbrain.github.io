maka hasil penjualan maksimal:
(10 x 1500) + (50 x 3000) = Rp. 15.000 + Rp. 150000
= Rp. 165.000

10. seorang pedagang mie ayam setiap membuat 50 porsi di perlukan biaya 270.000 jika dia menjual mie setiap satu porsi Rp 10.000 dan yang laku hanya 40 porsi tentukan untung atau ruginya​

Biaya membuat 50 porsi Mie = Rp270.000,00 (modal)

Maka biaya pembuatan 1 porsi mie = Rp270.000,00/50 = Rp5.400,00

Mie terjual 40 porsi

harga jual = Rp10.000,00

Hasil penjualan mie = 40 x Rp10.000,00 = Rp400.000,00

Hasil penjualan yang seharusnya didapat = Rp10.000,00 x 50 = Rp500.000,00

Keuntungan yang seharusnya didapat= Hasil penjualan -  modal  = Rp500.000,00 - Rp270.000 = Rp230.000,00

Akan tetapi karena yang terjual hanya 40, maka keuntungan = Rp400.000,00

- Rp270.000,00 = Rp130.000,00

Mie yang tidak terjual = (50-40) x Rp10.000,00 = Rp. 100.000,00 >> kerugian

Laba bersih Rp130.000,00  - Rp. 100.000,00 = Rp30.000,00

26. sebuah restoran menyediakan tiga jenis makanan favorit, yaitu mie goreng, sate rusa, dan nasi goreng. Keluarga firdaus membayar 380 ribu untuk 1 porsi mie goreng, 3 porsi sate rusa, 2 porsi nasi goreng, sedangkan keluarga syahrul membayar 520 ribu untuk 5 porsi sate rusa dan 2 porsi nasi goreng. Harga tiap porsi tetap dan harga 3 porsi mie goreng sama dengan harga 1 porsi nasi goreng. Jika x, y, dan z masing-masing mewakili harga setiap porsi mie goreng, sate rusa, dan nasi goreng a. buatlah model matematika masalah diatas, kemudian nyatakan dalam bentuk persamaan matriks b. berapakah harga tiap porsi ketiga jenis makanan itu


Model matematika pada soal ini adalah

Firdaus : x + 3y + 2z = 380.000

Syahrul : 5y + 2z = 520.000

Matriknya (terlampir pada gambar)

Harga masing-masing porsi ketiga jenis makanan adalah:

a. mie goreng Rp 20.000,-

b. sate rusa Rp 80.000,-

c. nasi goreng Rp 60.000,-

Sistem persamaan linear tiga variabel (SPLTV) adalah bentuk perluasan dari sistem persamaan linear dua variabel (SPLDV).

Ciri-ciri SPLTV adalah menggunakan relasi tanda sama dengan, memiliki 3 variabel, dan masing-masing variabel bersuku satu atau berpangkat satu.

Pembahasan

Diketahui:

Keluarga Firdaus menghabiskan Rp 380.000,- untuk 1 porsi mie goreng, 3 porsi sate rusa, dan 2 porsi nasi goreng.

Kelurga Syahrul menghabiskan Rp 520.000,- untuk 5 porsi sate rusa dan 2 porsi nasi goreng.

harga 3 porsi mie goreng = harga 1 porsi nasi goreng

Ditanya:

a. bagaimana model matematika dan persamaan dalam bentuk matriks?

b. berapa harga per porsi dari mie goreng, sate rusa, dan nasi goreng?

Jawab:

a. Model matematika dari soal tersebut.

Dimisalkan:

mie goreng = x

sate rusa = y

nasi goreng = z

Maka model matematikanya adalah

Keluarga Firdaus

x + 3y + 2z = 380.000 ........................(1)

Keluarga Syahrul

5y + 3z = 520.000 ..............................(2)

3 posi mie goreng = 1 porsi nasi goreng

3x = z  ......................................................(3)

Persamaan matriksnya (terlampir pada gambar)

b. Menentukan harga per porsi masing-masing jenis makanan

Untuk menentukan harga per porsi dari ketiga jenis makanan yang merupakan sistem persamaan 3 variabel, kita bisa menggunakan metode substitusi, metode eliminasi dan metode campuran substitusi - eliminasi.

1. Substitusi persamaan (3) ke persamaan (1) dan (2)

 x + 3y + 2z = 380.000

          x + 3y + 2(3x) = 380.000

          x + 3y + 6x = 380.000

          7x + 3y = 380.000  .................................(4)

 5y + 2z = 520.000

          5y + 2(3x) = 520.000

          5y + 6x = 520.000

          6x + 5y = 520.000 .......................................(5)

2. Eliminasi y dari persamaan (4) dan (5)

   7x + 3y = 380.000 ⇒ dikali 5 ⇒ 35x + 15y = 1.900.000

   6x + 5y = 520.000 ⇒ dikali 3 ⇒ 18x + 15y = 1.560.000

                                                         17x           = 340.000

                                                          x              = 340.000 : 17

                                                          x              = 20.000

Subtitusikan nilai x pada persamaan (3) maka diperoleh:

  3x = z ⇒ 3 x 20.000 = z

  z = 60.000

3. Setelah diperoleh nilai x = 20.000 dan z = 60.000 maka untuk mencari nilai y dapat dilakukan dengan substitusi nilai x dan z ke persamaan (1), sehingga diperoleh:

  x + 3y + 2z = 380.000

  20.000 + 3y + (2 x 60.000) = 380.000

  20.000 + 3y + 120.000 = 380.000

  3y = 380.000 - (20.000 + 120.000)

  3y = 380.000 - 140.000

  3y = 240.000

  y = 240.000 : 3

  y = 80.000

Jadi harga per porsi dari tiga jenis makanan adalah

a. mie goreng = Rp 20.000,-

b. sate rusa = Rp 80.000,-

c. nasi goreng = Rp 60.000,-
