Memainkan Kaset dan Recording pada Tape Recorder

Untuk memainkan kaset, ikuti langkah berikut:

1. Pertama, menempatkan kaset di tape recorder.
2. Kedua, mendorong tombol PLAY.
3. Selanjutnya, sesuaikan VOLUME.
4. Kemudian, tekan tombol STOP untuk berhenti.
5. Akhirnya, tekan tombol FAST FORWARD atau CEPAT BALIK untuk pergi ke bagian yang diinginkan.

Untuk merekam pada tape recorder Anda, ikuti langkah berikut:

1. Pertama, menempatkan kaset kosong di tape recorder.
2. Kedua, tekan tombol RECORD.
3. Kemudian, bicara ke arah mikrofon.
4. Ingat t menekan tombol rewind untuk mundur kaset.
5. Akhirnya, nanah PLAY untuk mendengarkan rekaman Anda.

Memutar sebuah kaset dan merekam di tape rekorder

Untuk memutar sebuah kaset, ikuti langkah- langkah berikut :
1. Pertama, letakkan kaset ke dalam tape rekorder
2. Kedua, tekan tombol PLAY (main)
3. Selanjutnya, atur volumenya
4. Lalu, tekan tombol STOP untuk berhenti.
5. Terakhir, tekan tombol FAST FORWARD (ke depan) atau FAST BACKWARD (ke belakang) untuk pergi ke bagian tertentu.

Untuk merekam di tape rekordermu, ikuti langkah-langkah berikut :
1. Pertama, letakkan kaset kosong ke dalam tape rekorder.
2. Kedua, tekan tombol RECORD (rekam)
3. Kemudian, berbicaralah dengan mikrofon.
4. Ingat untuk menekan tombol REWIND (putar balik) untuk memutar balik kasetnya.
5. Terakhir, tekan tombol PLAY untuk mendengar rekamanmu.
