1. Apa perbadaan antara blog pribadi dengan blog profesional


Jawaban:

Blog Pribadi

Konten blog pribadi biasanya sangat bebas, tidak terikat aturan tertentu. Pemilik blog bebas menentukan konten seperti apa yang akan ditulis. Bisa mengenai cerita liburan, pengalaman sekolah, cerita tentang keluarga, atau bahkan tentang kisah asmaranya.

Blog Profesional

Berbeda dengan blog pribadi, blog profesional memiliki batasan tertentu dalam konten yang diterbitkan. Biasanya blog profesional berisi konten-konten yang dapat menunjukkan profesionalitas pemilik blog di suatu bidang.

Misalnya, Anda ahli di bidang SEO dan digital marketing maka Anda bisa mengisi blog profesional Anda dengan konten yang berkaitan dengan SEO dan digital marketing.

27. tolong artikan cerita blog dalam bahasa indonesia


Belog
Disebuah desa di Bali hidup seorang anak yang konyol tapi bahagia yang bernama Belog. 'Belog' berarti 'Bodoh' dalam bahasa Bali, dan Belog terkenal di desanya karena hal-hal bodoh yang selalu ia lakukan. Warga di desa itu tidak pernah lelah menceritakan satu sama lain cerita tentang perbuatan Belog yang berpikiran simpel. Salah satu cerita favorit seperti berikut.
Satu hari ibu Belog tidak sengaja menjatuhkan korek api terakhirnya kedalam genangan air. korek api tersebut menjadi basah dan tidak bisa dinyalakan. Dia sekarang butuh korek api untuk menyalakan kompor, jadi dia memanggil putranya dan berkata, 'Belog tolong pergi ke toko dan beli beberapa korek api. Korek api yang kupunya basah, dan aku tidak bisa menyalakan api untuk memasak makanan kita.' dia memberikan Belog uang dan berkata " sekarang pastikan bahwa korek api yang kau beli kering dan dapat dinyalakan dengan mudah!"
Lalu Belog mengambil uang untuk membeli korek api dari ibunya, dan dengan bahagia berjalan ke toko. Dia masuk kedalam dan meminta sekotak korek api kepada pemilik toko. Lalu Belog dengan hati-hati menghitung uangnya dan membayar korek api tersebut lalu pulang kerumahnya.
Namun ketika dia diluar toko dia mendapat sebuah ide. Dia membuka kotak itu dan melihat kedalamnya. "korek api ini nampak bagus menurutku", pikirnya " tapi bagaimana bisa aku yakin? sebaiknya mencoba satu". Belog mengeluarkan satu dan menggeseknya. korek api itu menyala dengan mudah.
"Ah, korek api ini bagus," katanya "namun bagaimana dengan yang lainnya?" jadi Belog mengeluarkan korek api tersebut satu demi satu untuk untuk mencoba apakah korek api tersebut bagus lalu dia meletakkan kembali korek api tersebut kedalam kotak " ini sekotak korek api yang bagus" Belog berkata kepada dirinya lalu kembali kerumah.
" ini ibu" kata Belog, dengan senyum yang jujur dan konyol di mukanya "aku membeli sekotak korek api yang bagus. semuanya menyala, aku telah mencoba semuanya!" lalu Belog memberikan ibunya sekotak korek api yang telah dipakai dan dibakar
Ibunya yang malang. apa yang bisa ia katakan? lagipula Belog adalah anak satu-satunya, dan dia sangat mencintainya.
