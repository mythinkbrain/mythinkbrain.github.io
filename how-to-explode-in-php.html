3. cara menggunakan explode PHP untuk membuat tanggal kadaluarsa



  $expired_date = "2016-02-06";
  function is_expired($expired_date) {
  list($year, $month, $day) = explode('-', $expired_date);
  $new_expired_date = sprintf('%04d%02d%02d', $year, $month, $day);
  $date_now = date("Y-m-d");
    if ($new_expired_date > $date_now) {
    return false;
    } else {
    return true;
    }
  }

10. [tex]Quizz[/tex] eng?: Ratsu had Php 50.00 to spend on seven pencils. After buying, she had Php 8.00. How much did each pencil cost? Pakai cara! asal? report jan google Note: enak


✘ Diketahui:

Ratsu ingin membeli 7 pensil seharga 50.00 peso

Setelah membayar, uangnya sisa 8.00 peso

[tex]-----------[/tex]

✘ Ditanya:

Berapa harga yang dibutuhkan untuk setiap pensil?

[tex]-----------[/tex]

✘ Jawab:

7 pensil = 50.00 PHP

Setelah membayar, uangnya sisa 8.00 peso

7 pensil = 50.00 - 8.00

              = 42.00 PHP

[tex]-----------[/tex]

7 pensil = 42.00 PHP

1 pensil = 42.00 ÷ 7

            = 6 PHP (₱)

[tex]-----------[/tex]

✘ Jadi, harga untuk 1 buah pensil tersebut adalah sebesar ₱ 6.00 / 6
