23. Berat 4 kantong almon dan 3 kantong kacang tanah adalah 750g. Berat 3 kantong almon dan 5 kantong kacang tanah sama dengan berat 14 kantong kacang tanah. Berapa gram berat masing-masing kantong almon dan kantong kacang tanah?​


Jawaban:

Misalkan x adalah berat satu kantong almon dan y adalah berat satu kantong kacang tanah.

Kita dapat menyusun persamaan sebagai berikut:

4x + 3y = 750

3x + 5y = 14y

Untuk menyelesaikan persamaan di atas, kita bisa mengurangkan persamaan kedua dengan persamaan pertama:

4x + 3y - 3x - 5y = 750 - 14y

x - 2y = 750 - 14y

x = 750 - 14y + 2y

x = 750 - 12y

Selanjutnya, kita bisa menggunakan persamaan yang kedua untuk mencari nilai y:

3x + 5y = 14y

3(750 - 12y) + 5y = 14y

2250 - 36y + 5y = 14y

-31y = -750

y = -750/-31

y = 24,19 gram

Jadi, berat satu kantong kacang tanah adalah 24,19 gram. Selanjutnya, kita bisa menggunakan persamaan yang pertama untuk mencari berat satu kantong almon:

4x + 3y = 750

4x + 3(24,19) = 750

4x + 72,57 = 750

4x = 750 - 72,57

4x = 677,43

x = 677,43/4

x = 169,36 gram

Jadi, berat satu kantong almon adalah 169,36 gram.
