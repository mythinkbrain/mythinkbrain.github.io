Penjelasan dengan langkah-langkah

(x² + 1)(y² + 1) + 16 = 8(x + y)
⇔ x²y² + x² + y² + 1 + 16 = 8(x + y)
⇔ x²y² + (x + y)² – 2xy + 1 + 16 = 8(x + y)
⇔ x²y² – 2xy + 1 + (x + y)² – 8(x + y) + 16 = 0

x²y² – 2xy + 1 = (xy – 1)²(x + y)² – 8(x + y) + 16 = (x + y – 4)²

⇔ (xy – 1)² + (x + y – 4)² = 0
⇔ (xy – 1)² = –(x + y – 4)²

Untuk x,y ∈ ℝ:

xy – 1 = 0
⇒ xy = 1x + y – 4 = 0
⇒ x + y = 4

Nilai yang kita cari:
x³ + y³ = (x + y)³ – 3xy(x + y)
⇔ x³ + y³ = 4³ – 3·1·(4)
⇔ x³ + y³ = 64 – 12
∴  x³ + y³ = 52
