2. Harga dua hamburger dan satu paket kentang adlah 10. 000 sdangkan harga satu hamburger dan dua paket kentang adlah 8. 000 tentukan harga satu hamburger dan satu paket kentang tersebut?Please help me


Kelas 8 Matematika
Bab 4 - Sistem Persamaan Linear Dua Variabel

Misalkan
x = hamburger
y = paket kentang

2x + y = 10.000 .....(×2)
x + 2y = 8.000 .......(×1)

Eliminasi nilai y
4x + 2y = 20.000
x + 2y = 8.000
_____________ _
3x = 12.000
x = 12.000/3
x = 4.000

Substitusikan nilai x
2x + y = 10.000
2(4.000) + y = 10.000
y = 10.000 - 8.000
y = 2.000

x + y = 4.000 + 2.000
x + y = 6.000

Jadi harga satu hamburger dan satu paket kentang adalah Rp6.000,-

Tabungan awal = 25.000.000
Tabungan akhir = 26.500.000
Bunga(Rp) = Tab.Akhir - Tab.Awal
= 26.500.000 - 25.000.000
= Rp.1500.000

Dit:
Lama menabung atau LM= ?

Gunakan rumus ini:
Bunga(Rp)= LM/12 x % x Tab.Awal
1.500000= LM/12x18/100x25.000000
1.500000=LM/12 x 4.500.000
1.500.000 x 12 = LM x 4.500.000
18.000.000 = 4.500.000 LM
LM = 18.000.000 : 4.500.000
LM = 4 bulan

13. sebuah toko kelontong menjual buku pensil dan penghapus ketiganya dijual secara paket yang harganya ditentukan berdasarkan harga eceran nya masing-masing paket satu yang harganya Rp13.000 berisi buku pensil dan penghapus masing-masing 1 buah paket 2 berisi 3 buah buku 2 pensil dan sebuah penghapus harganya rp32.000 serta paket 3 berisi 2 buah buku sebuah pensil dan 3 buah penghapus harganya rp25.000 jika paket 4 yang berisi 5 buku 3 pensil dan 4 penghapus maka harga paket tersebut adalah​


Jika paket 4 berisi 5 buku, 3 pensil, dan 4 penghapus, maka harga paket tersebut adalah Rp.57.000.

Penjelasan :

Sistem persamaan linear tiga variabel atau yang biasa kita sebut dengan SPLTV adalah suatu persamaan matematika yang terdiri atas 3 persamaan linear yang masing - masing persamaannya memiliki tiga variabel (biasanya x, y dan z) dengan 1 penyelesaian. Sistem persamaan linear yang mencakup tiga variabel ini bisa digunakan untuk menentukan nilai harga satuan atau jumlah benda yang diwakilkan dalam bentuk variabel. Bentuk umum sistem persamaan linear tiga variabel adalah :

ax + by + cz = dex + fy + gz = hix + jy + kz = l

di mana a, e dan i adalah koefisien dari variabel x,

b, f dan j adalah koefisien dari variabel y,

c, g dan k adalah koefisien dari variabel z,

sedangkan d, h dan l adalah konstanta.

Himpunan penyelesaian suatu sistem persamaan linear tiga variabel (SPLTV) dapat ditentukan dengan beberapa cara, di antaranya adalah dengan menggunakan :

Metode subtitusiMetode eliminasiMetode gabungan atau campuranMetode determinanMetode invers matriks

Agar lebih jelasnya, simak pembahasan soal berikut.

PEMBAHASAN :

Ikuti alur pengerjaannya.

• Sebuah toko kelontong menjual buku, pensil, dan penghapus. Ketiganya dijual secara paket yang harganya ditentukan berdasarkan harga ecerannya masing - masing. Paket satu yang harganya Rp.13.000 berisi buku, pensil, dan penghapus masing - masing 1 buah.

Asumsikan harga buku sebagai B, harga pensil sebagai PL dan harga penghapus sebagai PG. Berarti, persamaan pertama yang dapat kita baca adalah B + PL + PG = 13.000.

• Paket 2 berisi 3 buah buku, 2 pensil, dan sebuah penghapus harganya Rp.32.000.

Berarti, persamaan kedua yang dapat kita baca adalah 3B + 2PL + PG = 32.000.

• Serta paket 3 berisi 2 buah buku, sebuah pensil, dan 3 buah penghapus harganya Rp.25.000.

Berarti, persamaan ketiga yang dapat kita baca adalah 2B + PL + 3PG = 25.000.

• Eliminasi persamaan 1 dan 2 untuk menghasilkan persamaan 4.

B + PL + PG = 13.000 [dikalikan 3]

3B + 2PL + PG = 32.000 [dikalikan 1]

menjadi

3B + 3PL + 3PG = 39.000

3B + 2PL + PG = 32.000   -

PL + 2PG = 7.000..............persamaan 4

• Eliminasi persamaan 1 dan 3 untuk menghasilkan persamaan 5.

B + PL + PG = 13.000 [dikalikan 2]

2B + PL + 3PG = 25.000 [dikalikan 1]

menjadi

2B + 2PL + 2PG = 26.000

2B + PL + 3PG = 25.000  -

PL - PG = 1.000..............persamaan 5

• Eliminasi persamaan 4 dan 5 untuk menentukan harga penghapus.

PL + 2PG = 7.000

PL - PG = 1.000   -

3PG = 6.000

PG = harga penghapus = 2.000

• Substitusikan harga penghapus ke persamaan 5 untuk menentukan harga pensil.

PL - PG = 1.000

PL - 2.000 = 1.000

PL = harga pensil = 3.000

• Substitusikan harga penghapus dan pensil ke persamaan pertama untuk menentukan harga buku.

B + PL + PG = 13.000

B + 3.000 + 2.000 = 13.000

B + 5.000 = 13.000

B = harga buku = 8.000

• Jika paket 4 berisi 5 buku, 3 pensil, dan 4 penghapus, maka harga paket tersebut adalah :

5B + 3PL + 4PG

= 5(8.000) + 3(3.000) + 4(2.000)

= 40.000 + 9.000 + 8.000

= Rp.57.000
