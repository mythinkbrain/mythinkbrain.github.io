10. Amir naik kereta api dari jakarta ke surabaya , pertanyaan berikut ini benar,kecuali A. Amir diam terhadap kursi tempat duduknya B. Amir diam terhadap kereta C. Kereta api bergerak terhadapa stasiun D. Amir bergerak terhadap masinis


Salam sejahtera,

Pernyataan berikut yang benar kecuali D. Amir bergerak terhadap masinis.

Pembahasan ;

  Amir bergerak terhadap masinis itu salah, masinis adalah orang yang menggerakkan kereta api atau masinis adalah juru mesin dalam kereta api. Posisi Amir masuk ke dalam kereta api adalah diam, meskipun Amir diam namun posisi Amir berpindah karena kereta api melaju. Jalannya kereta api tentu ke arah stasiun di tuju.

Mari membahas pilgannya.

pilihan A

  Disebut benar karena Amir memang diam di tempat duduknya saat kereta berjalan

Pilihan B

  Disebut benar karena Amir memang diam

Pilihan C

  Disebut benar karena kereta memang bergerak menuju stasiun dituju

Pilihan D

  Disebut tidak benar, masinis berada posisinya paling depan kereta sebagai juru stasiun dan Amir diam terhadap tempat duduknya di kereta

kesimpulan paling tepat yang tidak benar adalah D.

19. Pada rangkaian gerbong kereta api perbandingan tempat duduk ekonomi dan bisnis adalah 1:3 banyak tempat duduk rangkaian kereta api tersebut 480 tempat duduk. Jika harga tiket ekonomi rp 80.000,00 dan harga tiket bisnis rp 120.000,00. Tentukan perolehan penjualan tiket penumpang jika tempat duduk kereta api terisi penuh penumpang!​


Jawaban:

Rp 52.800.000

Penjelasan dengan langkah-langkah:

Tiket ekonomi = 1/4 × 480

= 120 kursi

= 120 × Rp 80.000

= Rp 9.600.000

Tiket bisnis = 3/4 × 480

= 360 kursi

= 360 × Rp 120.000

= Rp 43.200.000

Perolehan penjualan = tiket ekonomi + tiket bisnis

= Rp 9.600.000 + Rp 43.200.000

= Rp 52.800.000

Jadi perolehan penjualan tiket Rp 52.800.000

semoga membantu

Jawab:

Diketahui perbandingan penumpang kereta 1:3 ekonomi:bisnis

Penumpang ekonomi+bisnis= 480

Banyaknya penumpang ekonomi=

1/4x480= 120 penumpang.

Banyaknya penumpang bisnis=

3/4x480= 360 penumpang.

Jika seluruh tiket habis maka:

Tiket ekonomi = 120x Rp. 80.000= 9.600.000

Tiket bisnis = 360x Rp. 120.000= 43.200.000

Total penjualan tiket: Rp. 52.800.000
