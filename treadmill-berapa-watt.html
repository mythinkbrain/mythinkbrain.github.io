18. Sebuah trafo mempunyai daya sekunder 8000 watt. Jika efisiensi trafo 20%. Daya primer trafo adalah …. *400 watt40.000 watt40 watt4000 watt​


Penjelasan:

Ps = daya sekunder

Pp = daya primer

efisiensi = Ps/Pp x 100%

Pp = (100 % / efisiensi) x Ps

Pp = (100% / 20%) x 8000

Pp = 5 x 8000

Pp = 40.000 watt
