5. kalo processor Intel(R) Celeron(R) N4120 CPU @ 1.10GHz (4CPUs), 1.1GHz level nya setara atau lebih tinggi dari processor Core i3?​


Penjelasan:

Processor Intel Celeron N4120 merupakan processor kelas bawah yang umumnya dianggap kurang powerful dibandingkan dengan processor Core i3. Processor ini memiliki jumlah core yang lebih sedikit, kecepatan clock yang lebih rendah, dan fitur yang kurang canggih dibandingkan dengan banyak processor Core i3. Processor ini umumnya cocok untuk tugas-tugas dasar seperti browsing web, mengedit dokumen, dan memutar media ringan, tetapi mungkin kesulitan dalam menangani beban kerja yang lebih berat seperti gaming atau editing video. Secara umum, processor Core i3 dianggap sebagai peningkatan performa dibandingkan dengan Celeron N4120.

9. Mengapa Intel mengeluarkan pentium 2 Celeron jelaskan​


Jawaban:

Celeron.jpeg Celeron2.jpg Celeron3.jpg

Celeron adalah keluarga mikroprosesor buatan Intel Corporation dengan arsitektur Pentium II, tetapi dengan sedikit pengurangan memori cache demi mempertahankan faktor ekonomis. Prosesor ini resmi dirilis ke pasar pada bulan April 1998. Memori cache L2 Celeron hanya berukuran 128 KB, sementara Pentium II memiliki memori cache L2 sebesar 512 KB. Celeron dapat dipasang pada papan induk yang mempunyai slot 1 maupun soket PGA 370. Pada bulan Januari 2008, Intel Corporation memperbarui jajaran prosesor Celeron mereka dengan teknologi terbaru, berbasiskan Core 2 dengan kode core Allendale, tetapi dengan cache sebesar 25% jika dibandingkan dengan prosesor Core 2.

dik x = i3, ->> 5000.000

ok pertama tama, marilah kita mengucapkan syukur kepada Tuhan YME.haha

5000.000x +7500.000y ≤ 400.000.000 (sederhanakan dibagi 1000.000)

so , 5x +7.5 y  ≤ 400

7.5 y  ≤ 400

Eliminasi x

dari ketigak persamaan diatas cari keuntungan maksimal yang di dapat

persamaan pertama x,y = 80,53

persamaan kedua x,y = 60,60

persamaan ketiga x,y = 20,40

dari ketiga persamaan diatas didapat nilai x dan y. kemudian cari keuntungan masing masing x dan y nya.

titik 1 x,y = 0,53

keuntungan = 500.000x +500.000y

keuntungan = 0 + 500.000* 53

titik 2 x,y = 20,40

keuntungan = 500.000x +500.000y

keuntungan = 500.000*20 +500.000*40

keuntungan = 30.000.000

titik 3 x,y = 60,0

keuntungan = 500.000x +500.000y

keuntungan = 500.000*60 +0

keuntungan = 30.000.000

jadi keuntungan maksimal bisa kita lihat pada data diatas. yaitu 30.000.000

28. mengapa intel mengekuarkan pentium 2 celeron,jelaskan​


Jawaban:

Celeron adalah keluarga mikroprosesor buatan Intel Corporation dengan arsitektur Pentium II, tetapi dengan sedikit pengurangan memori cache demi mempertahankan faktor ekonomis. Prosesor ini resmi dirilis ke pasar pada bulan April 1998.

Jawaban:

Untuk menyaingi pasar dagang CPU, terutama dibagian Workstation atau Server.

Penjelasan:

Pentium II sebenarnya sama seperti Pentium Pro, dan prosesor generasi keenam dari keluarga P6 lainnya. Akan tetapi, desainnya yang agak radikal membuatnya menjadi pembeda. Dengan menggunakan teknologi 350 nanometer (0.35 mikron) dan 250 nanometer (0.25 mikron) dan dilengkapi dengan instruksi MMX, prosesor ini menjadi prosesor untuk mainstream setelah Pentium MMX, setelah Pentium Pro mengalami kegagalan pada kelas desktop dan laku hanya pada server.
