4. Bilangan yang seharusnya dihilangkan pada barisan 3,6,9,12,20 adalah... pakai cara ya:)


bilangan yang seharusnya dihilangkan adalah bilangan 20
karena
3,6,9,12,...
diketahui bahwa
suku awal (a) = 3
selisih tiap suku (b) = 6-3 = 3
                             = 9-6 = 3
                             = 12-9 = 3
maka haruslah a=3,b=3
angka 20 tak memenuhi3+3=6 , 6+3=9 , 9+3= 12 , 12+3 =15
jadi yg di buang adalah 20 karna bilangan 3,6,9,12 bilangan yg di tambah 3

16. bagaimana cara kerja admin server


Seoarang admin server tidak bekerja sendirian, biasanya mereka membentuk sebuah team. Sangat mustahil hanya terdapat 1 admin server dalam sebuah perusahaan besar, dikarenakan menjadi admin server itu sangatlah sulit dan beresiko.

Seorang admin server tidak hanya bekerja dengan rekannya saja. Mereka juga bekerja sama dengan pekerja lainnya seperti supervisor, hingga pekerja teknik. Selain itu, sebagai seorang admin server harus pandai berbicara dikarenakan admin server juga harus dapat menjelaskan berbagai hal-hal teknis kepada orang awam. Admin server merupakan orang yang mengatur semua aktifitas server, mulai dari awal dibangun, perawatan sampai perbaikan.

Perlu anda ketahui bahwa menjadi admin server bukan perkara yang mudah. Admin server dituntut untuk merawat dan menjaga kinerja server agar tetap berjalan dengan baik. Dalam sebuah perusahaan, peran admin server sangatlah penting.

Admin server mempunyai 3 tugas utama seperti membangun, mengelola, dan memperbaiki server. Ketiga tugas itulah yang “wajib” dilakukan seorang admin server untuk menjaga server tetap berjalan dengan baik.

Barisan adalah susunan bilangan yang memiliki pola tertentu. Sedangkan penjumlahan suku-suku terurut pada barisan disebut dengan deret. Yukk langsung aja kita lihat penjabaran jawaban soal kali ini

Diketahui kumpulan bilangan sebagai berikut : 8, 12, 15, 18, 20, 21, 24.

Yang paling mungkin adalah dengan membawa kumpulan bilangan di atas menjadi barisan aritmatika.

Barisan aritmetika adalah barisan bilangan di mana setiap dua suku yang berurutan memiliki selisih atau beda yang tetap.

Kumpulan bilangan 8, 12, 15, 18, 20, 21, 24, agar dapat menjadi barisan maka bilangan yang harus dihilangkan adalah 8 dan 20 sehingga menjadi 12, 15, 18, 21, 24.

Pada barisan 12, 15, 18, 21, 24, selisih antar bilangannya adalah sama yaitu 3 sehingga disebut sebagai barisan aritmatika.
