1. Sebuah kotak perhiasan berbentuk balok berukuran 20 cm x 12 cm x 8 cm. Bagian luar kotak tersebut akan dilapisi kain beludru. Jika harga beludru Rp 30.000 per meter, maka biaya yang diperlukan adalah...​


Jawaban:

Jadi biaya yang dikeluarkan adalah Rp 2.976

Penjelasan dengan langkah-langkah:

Diketahui

Kotak

p = 20 cm

l = 12 cm

t = 8 cm

beludru per meter = 30.000

Ditanya Biaya yang diperlukan?

Jawab

L = 2 x ( p x l + p x t x l x t )

L = 2 x ( 20 x 12 + 20 x 8 + 12 x 8 )

L = 2 x ( 240 + 160 + 96)

L = 2 x 496

L = 992 cm² = 0,0992 m²

Biaya yang dikeluarkan

= 0,0992 x 30.000

=2.976

Jadi biaya yang dikeluarkan adalah Rp 2.976

Jawaban:

biaya kain beludru : Rp. 2.976

Penjelasan dengan langkah-langkah:

diketahui :

Panjang (P) = 20 cm

Lebar (L) = 12 cm

Tinggi (T) = 8 cm

Luas Balok

L = 2 x ( P x l) + (L x t) + (P x t)

= 2 x (20 x 12) + (12 x 8) + (20 x 8)

= 2 x ( 240) + ( 96 ) + 160

= 2 x 496

L = 992 cm²

L = 992 cm² = 0,0992 m²

______

10.000

biaya yang diperlukan :

ukuran kotak balok x harga kain beludru

0,0992 m² x Rp. 30.000

= Rp. 2.976

Semoga bermanfaat
