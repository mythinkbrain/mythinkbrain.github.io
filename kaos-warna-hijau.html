9. 1. Warna biru : warna hijau = 3 : 4, jika warna biru sebanyak 54 buah, maka warna hijau = ...2. Kaos Merah : Kaos Putih = 3 : 4, Jika Kaos Putih Sebanyak 52 Buah, maka kaos merah = ...3. Jaket merah : Jaket Biru = 3 : 5, Jika Jaket biru sebanyak 35 buah, maka jumlah semua jaket = ...​


1) Biru : Hijau = 3 : 4

Merah = 3/4 x 52 buah

3) Merah : Biru = 3 : 5

Biru = 35 buah

Merah = 3/5 x 35 buah

Merah = 21 buah

Jumlah semua jaket = 35 buah + 21 buah

Jumlah semua jaket = 56 buah

40x+50x=360
90x=360
x=4

maka total kaos yg terjual 12+4=16.
