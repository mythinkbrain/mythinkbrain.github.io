21. Kuis Spesial Syukuran Pacar Baru. Nama Pacar saya adalah y = 4/3x + 47/3. Gradien Garis Pacar saya berapa?Note = Jangan langsung jawap ya make penjelasan. nanti pacar saya marah kalau langsung jawab.​


Jawaban:

gradien (m) = 4/3

Penjelasan dengan langkah-langkah:

Bentuk persamaan garis di atas sama dengan bentuk y = mx + c

Untuk bentuk y = mx + c, gradiennya adalah koefisien x (depan nilai x)

Jadi, gradien y = 4/3x + 47/3 adalah koefisien x yaitu 4/3

Jawaban:

4/3

Penjelasan dengan langkah-langkah:

y = 4/3x + 47/3

Rumus;

y = mx + c

Gradien = m

Maka*

y = mx + c

y = 4/3x + 47/3

m = 4/3

terimakasih
