Matriks adalah susunan bilangan atau simbol yang disusun dalam bentuk baris dan kolom sehingga membentuk suatu bangun persegi.

Jenis –jenis Matriks

• Matriks Baris

adalah matriks yang hanya terdiri satu baris.

• Matriks Kolom

adalah matriks yang hanya terdiri satu kolom.

• Matriks Nol

adalah matriks yang semua elemennya berupa bilangan nol.

• Matriks Persegi

Matriks ini biasanya terjadi ketika jumlah baris sama dengan jumlah kolom.

• Matriks Diagonal

adalah matriks persegi dengan bilangan pada posisi diagonalnya tidak nol.

29. Terdapat Array : A [2][2] maka jumlah elemen Array tersebut adalah …


Terdapat Array: A[2][2] maka jumlah elemen array tersebut adalah 4 elemen. Untuk mengetahui jumlah elemen pada array dua dimensi tersebut, kita dapat menganggap array tersebut sebagai sebuah matriks, dimana dengan ukuran 2x2 maka, akan didapat empat elemen didalam matriks.

Pembahasan

Array sendiri adalah suatu struktur data yang menyimpan sekelompok elemen dimana setiap elemen akan ditandai dengan suatu index. Array dapat berupa array satu dimensi, dua dimensi, maupun multi dimensi. Dalam konteks soal ini, array yang digunakan adalah array dua dimensi. Array dua dimensi dapat diidentifikasi dengan memiliki dua index penanda seperti, alamat suatu titik pada diagram kartesius. Bentuk umum dari array dua dimensi:

TipeData Variabel [x][y]

Contoh:

int contohMatriks [4][3]

Tipe data dapat berupa int, float, char, dan sebagainyaVariabel merupakan penamaan array yang disesuaikan kebutuhanX jika dianalogikan dalam matriks adalah jumlah barisY jika dianalogikan dalam matriks adalah jumlah kolom

Sehingga dalam array: A[2][2], maka memiliki dua baris dan dua kolom, dengan index:

A[0][0]   A[0][1]

A[1][0]    A[1][1]
