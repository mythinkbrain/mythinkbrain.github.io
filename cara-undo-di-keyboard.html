4. - Menghapus teks sebelah kiri menggunakan Tombol Backspace
    - Menghapus teks sebelah kanan menggunakan Tombol Delete

30. ctrl+A adalah kombinasi tombol keyboard yang sama artinya dengan ... A. clear B. select All C. undo D. redo


Jawaban:

Ctrl+A adalah kombinasi tombol keyboard yang sama artinya dengan B. Select All.

Adapun A merupakan singkatan dari All atau semua. Sementara tombol Ctrl merupakan salah satu tombol pemberi instruksi selain tombol Shift.

Penjelasan:

Ctrl+A merupakan salah satu kombinasi tombol yang digunakan aplikasi. Dengan tombol tersebut, pengguna tidak perlu mencari perintah Select All dari menu tab. Hal ini berdampak langsung pada semakin efisiennya waktu yang dibutuhkan dalam melakukan suatu kegiatan dalam kerangka aplikasi.
