4. 2.Seorang pengusaha toko tas sedang berbelanja tas,harga sebuah tas jinjing 25.000 sedangkan sebuah tas ransel 50.000 rp .Modal yang tersedia adalah 1.500.000 .kapasitas toko hanya mampu memuat 80 tas.Jika tas tersebut di jual kembali dan mendapat laba untuk tas jinjing 5000 dan tas ransel 10.000 tentukan model matematikanya!


Jawab: Model matematikanya adalah x + y ≤ 80 ; x + 2y = 60 ; x ≥ 0 dan y ≥ 0 dengan fungsi tujuan f(x,y) = 5.000x + 10.000y.

Penjelasan dengan langkah-langkah:

Dimisalkan tas jinjing adalah x dan tas ransel adalah y, model matematika yang terbentuk :

(a) Kapasitas toko hanya mampu memuat 80 tas : x + y ≤ 80.

(b) Modal yang dimiliki Rp1.500.000, maka : 25.000x + 50.000y = 1.500.000 bisa disederhanakan dengan masing-masing dibagi 25.000 ➡ x + 2y = 60.

(c) Keuntungan yang diinginkan : f(x,y) = 5.000x + 10.000y.

karena x dan y merupakan jumlah tas, berarti nilai x dan y adalah bilangan real, yaitu bernilai positif dan bilangan bulat : x ≥ 0 dan y ≥ 0.

Limbah tekstil dapat berupa sisa benang pada kelos, sisa potongan kain, dan bahan serta limbah cair dari pewarnaan tekstil.
