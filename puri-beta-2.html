Akan dibuktikan bahwa [tex](\cos 2 \beta + \cos 4 \beta + \cos 6 \beta) \sin \beta = \sin 3 \beta \cos 4 \beta[/tex]

Akan dicari bentuk sederhana dari [tex](\cos 2 \beta + \cos 4 \beta + \cos 6 \beta) \sin \beta[/tex] yaitu

[tex]= (\cos 2 \beta + \cos 4 \beta + \cos 6 \beta) \sin \beta\\= \cos 2 \beta \sin \beta + \cos 4 \beta \sin \beta + \cos 6 \beta \sin \beta\\= \frac{1}{2} (\sin (2 \beta+\beta) - \sin (2 \beta-\beta)) + \frac{1}{2} (\sin (4 \beta+\beta) - \sin (4 \beta-\beta)) + \frac{1}{2} (\sin (6 \beta+\beta) - \sin (6 \beta-\beta))[/tex]

[tex]= \frac{1}{2} (\sin 3 \beta - \sin \beta) + \frac{1}{2} (\sin 5 \beta - \sin 3 \beta) + \frac{1}{2} (\sin 7 \beta - \sin 5 \beta)\\= \frac{1}{2} (\sin 3 \beta - \sin \beta + \sin 5 \beta - \sin 3 \beta + \sin 7 \beta - \sin 5 \beta)\\= \frac{1}{2} (\sin 7 \beta - \sin \beta)[/tex]

Akan dicari bentuk sederhana dari [tex]\sin 3 \beta \cos 4 \beta[/tex] yaitu

[tex]= \sin 3 \beta \cos 4 \beta\\= \cos 4 \beta \sin 3 \beta\\= \frac{1}{2} (\sin (4 \beta+3 \beta) - \sin (4 \beta-3 \beta))\\= \frac{1}{2} (\sin 7 \beta - \sin \beta)[/tex]

Jadi terbukti diperoleh

[tex](\cos 2 \beta + \cos 4 \beta + \cos 6 \beta) \sin \beta = \sin 3 \beta \cos 4 \beta[/tex]

29. buktikan identitas trigonometri berikut[tex] \frac{ 2 \sin( \beta ) - \sin(2 \beta ) }{2 \sin( \beta ) + \sin(2 \beta ) } = { \tan( \frac{1}{2} \beta ) }^{2} [/tex]​


buktikan  identitas

[tex]\rm \dfrac{2 sin~ B - sin (2B)}{2 sin~B + sin(2B) } = \{tan (\frac{1}{2}B\}^2[/tex]

karena sin 2B  =  2 sin  B cos B

maka ruas kiri =

[tex]\rm = \dfrac{2sin~B - 2 sin~B . cos ~B}{2sin~B + 2 sin~B.cos~B}[/tex]

[tex]\rm = \dfrac{2sin~B (1 - cos ~B)}{2sin~B (1 + cos~B)}[/tex]

karena cos B =  1 - 2 sin² ¹/₂ B

maka

[tex]\rm = \dfrac{2sin~B (1 - (1- 2sin^2 \frac{1}{2}~B)}{2sin~B (1 +(1- 2sin^2 \frac{1}{2}~B)}[/tex]

[tex]\rm = \dfrac{2sin~B (1 - 1 + 2sin^2 \frac{1}{2}~B)}{2sin~B (1 +1 - 2sin^2 \frac{1}{2}~B)}[/tex]

[tex]\rm = \dfrac{2sin~B (2sin^2 \frac{1}{2}~B)}{2sin~B (2-2sin^2 \frac{1}{2}~B)}[/tex]

[tex]\rm = \dfrac{4sin~B . sin^2 \frac{1}{2}~B)}{2sin~B \{2(1 -sin^2 \frac{1}{2}~B)\}}[/tex]

[tex]\rm = \dfrac{4sin~B . sin^2 \frac{1}{2}~B)}{4sin~B (1 -sin^2 \frac{1}{2}~B)}[/tex]

[tex]\rm = \dfrac{sin^2 \frac{1}{2}B}{cos^2 \frac{1}{2}B}[/tex]

[tex]\rm = \{tan \frac{1}{2}B\}^2[/tex]

terbukti sama dengan ruas kanan


30. jika diketahui alpha dan beta merupakan akar dari x^2 + x - 3= 0 tentukan alpha^2 + beta^2 dan alpha/beta + beta/alpha


x^2 + x - 3 = 0
(x - 1)(x + 2) = 0
x = 1 atau x = -2
alfa = 1, beta = -2

alfa^2 + beta^2
= (1)^2 + (-2)^2
= 1 + 4
= 5

alfa/beta + beta/alfa
= 1/-2 + (-2/1)
= - 1/2 - 4/2
= - 5/2
