8. Ketika berdiri di perempatan jalan, budi mendengar frekuensi 510Hz, dari sirine mobil ambulan yang mendekatinya. setelah mobil ambulan melewati budi, budi mendengar frekwensi sirine 450 Hz . jika cepat rambat bunyi di udara 320m/s tentukan jarak kecepatan mobil ambulan


Jika cepat rambat bunyi di udara 320 m/s , maka kecepatan mobil ambulan adalah 20 m/s

PEMBAHASAN

Jika terjadi gerak relatif antara sumber bunyi dengan pendengar maka akan di hasilkan suatu gejala yang dinamakan Efek Doppler. Gejala ini menghasilkan frekuensi yang didengar oleh seorang pengamat berbeda dengan frekuensi sumber bunyi.

fp = ( v ± vp ) / ( v ± vs ) x fs

dimana :

fp = frekuensi yang didengar oleh pendengar ( Hz )

v = kecepatan bunyi ( m/s )

vp = kecepatan pendengar ( m/s )

vs = kecepatan sumber ( m/s )

fs = frekuensi sumber ( Hz )

Okay mari kita gunakan rumus ini untuk menyelesaikan soalnya.

Diketahui:

Frekuensi Mendekat = fp₁ = 510 Hz

Frekuensi Menjauh = fp₂ = 450 Hz

Cepat Rambat Bunyi = v = 320 m/s

Ditanyakan:

Kecepatan Mobil Ambulan = vs = ?

Penyelesaian:

Saat Mobil Ambulan Mendekat , maka :

fp₁ = ( v ) / ( v - vs ) x fs

510 = 320 / ( 320 - vs ) x fs

fs = 510 ( 320 - vs ) / 320

fs = ( 51/32 ) ( 320 - vs ) → Persamaan 1

Saat Mobil Ambulan Menjauh , maka :

fp₂ = ( v ) / ( v - vs ) x fs

450 = 320 / ( 320 + vs ) x fs

fs = 450 ( 320 + vs ) / 320

fs = ( 45/32 ) ( 320 + vs )

fs = ( 45/32 ) ( 320 + vs )

( 51/32 ) ( 320 - vs ) = ( 45/32 ) ( 320 + vs ) ← Persamaan 1

51 ( 320 - vs ) = 45 ( 320 + vs )

16320 - 51 vs = 14400 + 45 vs

16320 - 14400 = 51 vs + 45 vs

1920 = 96 vs

vs = 1920 / 96

vs = 20 m/s

17. 1. Sebuah mobil ambulan membunyikan suara sirinenya dengan frekuensi 400 Hz. Jika laju mobil ambulan itu 20 m/s, dan laju bunyi di udara saat itu 340 m/s, Maka tentukan frekuensi sirine yang didengar oleh orang saat itu ?​


Jawaban:

fp = 425 Hz

Penjelasan:

Misalnya suatu sumber bunyi yang bergerak mengeluarkan bunyi dengan frekuensi tertentu, maka frekuensi yang didengar oleh pengamat diatur dalam efek doppler. Secara umum efek doppler dirumuskan dengan:

[tex]\boxed{f_p=f_s\frac{v\pm v_p}{v\mp v_s}}[/tex]

[tex]\textbf{Dimana:}[/tex]  

[tex]f_p=\textrm{frekuensi yang didengar pengamat }\mathrm{ (Hz)}[/tex]  

[tex]f_s=\textrm{frekuensi sumber }\mathrm{ (Hz)}[/tex]  [tex]v=\textrm{kecepatan bunyi di udara }\mathrm{ (m/s)}[/tex]

[tex]v_p=\textrm{kecepatan pengamat }\mathrm{ (m/s)}[/tex]

[tex]v_s=\textrm{kecepatan sumber bunyi }\mathrm{ (m/s)}[/tex]

Diketahui:

[tex]$\begin{align}&f_s=400\ \mathrm{Hz}\\&v_s=20\ \mathrm{m/s\\&v=340\ \mathrm{m/s}}\\&v_p=0\ \mathrm{m/s}\end{align}[/tex]

Ditanyakan:

[tex]f_p=...?[/tex]

Langkah Penyelesaian:

[tex]$\begin{align}&f_p=f_s\frac{v\pm v_p}{v\mp v_s}\\&f_p=400\ \mathrm{Hz}\times\frac{340\ \mathrm{m/s}}{340\ \mathrm{m/s}-20\ \mathrm{m/s}}\\&f_p=425\ \mathrm{Hz}\end{align}[/tex]

22. Pernahkah kamu berdiri di trotoar jalan kemudian mendengar suara sirine mobil ambulan? Saat mobil berada di kejauhan bunyi sirene mobil terdengar pelan. Ketika mobil bergerak mendekati kamu, bunyi akan terdengar semakin keras kemudian bunyi sirine mobil akan terdengar melemah lagi setelah mobil menjauh. Dapatkah kamu menjelaskan mengapa bunyi sirine mobil berubah-ubah ketika mobil bergerak mendekati/menjauhi kamu?


Jawaban:

Frekuensi sirine berubah - ubah karena adanya efek doppler

Penjelasan:

Apa itu Efek Doppler?

Efek Doppler adalah efek yg menjelaskan tentang hubungan antara jarak pengamat terhadap sirine dengan frekuensi bunyi

Bunyinya :

"Apabila pengamat mendekati sumber bunyi, maka frekuensi yang didengar semakin besar. Apabila pengamat menjauhi sumber bunyi, maka frekuensi yang didengar semakin kecil"

Apa saja contoh dari Efek Doppler?

Contohnya :

Ketika mobil kebakaran sedang melaju. Pada awalnya suara sirine-nya kecil. Semakin mendekati kita suaranya makin besar. Kemudian mobil kebakaran menjauh dan suara sirine-nya mengecil kembali.

Rumus Efek Doppler apa sih?

[tex]fp = \frac{v \pm vp}{v \pm vs} \times fs[/tex]

Dimana :

fp : Frekuensi pengamat (Hz)

v : Cepat rambat bunyi di udara (m/s)

vp : Kecepatan pengamat [jika bergerak] (m/s)

vs : Kecepatan sirine [jika bergerak] (m/s)

fs : Frekuensi sumber bunyi (Hz)

Tanda ± di atas dapat berarti + ataupun - . Tergantung si pendengar atau si sumber bunyi bergerak atau tidak. Aturannyan :

vp bernilai + jika si pendengar mendekati sumber suara, dan bernilai – jika menjauhi sumber suara

vs bernilai + jika sumber suara menjauhi pendengar, dan bernilai jika mendekati pendengar

25. sebuah mobil ambulan membunyikan suara sirine dengan frekuensi 400hz jika laju mobil ambulan itu 20 m/s dan laju bunyi di udara saat itu 340m/s maka tentukan frekuensi sirine yang didengar orang saat itu​


Jawaban:

fp = 425 Hz

Penjelasan:

Misalnya suatu sumber bunyi yang bergerak mengeluarkan bunyi dengan frekuensi tertentu, maka frekuensi yang didengar oleh pengamat diatur dalam efek doppler. Secara umum efek doppler dirumuskan dengan:

[tex]\boxed{f_p=f_s\frac{v\pm v_p}{v\mp v_s}}[/tex]

[tex]\textbf{Dimana:}[/tex]  

[tex]f_p=\textrm{frekuensi yang didengar pengamat }\mathrm{ (Hz)}[/tex]  

[tex]f_s=\textrm{frekuensi sumber }\mathrm{ (Hz)}[/tex]  [tex]v=\textrm{kecepatan bunyi di udara }\mathrm{ (m/s)}[/tex]

[tex]v_p=\textrm{kecepatan pengamat }\mathrm{ (m/s)}[/tex]

[tex]v_s=\textrm{kecepatan sumber bunyi }\mathrm{ (m/s)}[/tex]

Diketahui:

[tex]$\begin{align}&f_s=400\ \mathrm{Hz}\\&v_s=20\ \mathrm{m/s\\&v=340\ \mathrm{m/s}}\\&v_p=0\ \mathrm{m/s}\end{align}[/tex]

Ditanyakan:

[tex]f_p=...?[/tex]

Langkah Penyelesaian:

[tex]$\begin{align}&f_p=f_s\frac{v\pm v_p}{v\mp v_s}\\&f_p=400\ \mathrm{Hz}\times\frac{340\ \mathrm{m/s}}{340\ \mathrm{m/s}-20\ \mathrm{m/s}}\\&f_p=425\ \mathrm{Hz}\end{align}[/tex]
