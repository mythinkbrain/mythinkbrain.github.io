Berapa dus keramik yang dibutuhkan untuk menutup lantai dan biaya yang dibutuhkan untuk membeli keramik merk Mulia tersebut?


Banyaknya dus keramik yang diperlukan untuk dipasang pada lantai Pak Abi adalah 134 kardus.

Biaya yang dibutuhkan sebesar Rp. 8.710.000,00.

Penjelasan dengan langkah-langkah:

Luas bangun datar adalah besarnya ruang yang ada pada bangun datar tersebut. Pada persegi untuk menghitung luasnya dapat mengalikan panjang sisinya. Sedang pada persegi panjang dapat mengalikan panjang dan lebarnya. Pada bangun segitiga dapat mengalikan alas dan tinggi segitiga kemudian dibagi 2, dan masih banyak lagi rumus luas yang lainnya.

Pada lantai Pak Abi dengan luas tersebut akan dipasang keramik. Keramik ini memiliki luas tertentu. kemudian dapat diketahui jumlah keramik yang dibutuhkan untuk menutupi lantai tersebut. Setelah itu untuk mengetahui jumlah kardus yang diperlukan dapat dihitung dengan membagikan keramik yang dibutuhkan dengan jumlah keramik dalam satu dus. Untuk mengetahui biaya dapat mengalikan jumlah kardus dengan harga tiap kardusnya.

Diketahui : P lantai = 16 m

                  L lantai = 8 m

                  Ukuran keramik = 40 cm x 40 cm

                  1 dus berisi 6 pcs seharga Rp. 65.000,00

Ditanya : Berapa dus kebutuhan keramik dan biaya yang dibutuhkan?

Jawab :

Luas lantai = P x L

                  = 16 x 8

                  = 128 [tex]m^{2}[/tex]

Luas keramik = s x s

                      = 0,4 m x 0,4 m

                      = 0,16 [tex]m^{2}[/tex]

Kebutuhan keramik = Luas lantai : luas keramik

                                 = 128 : 0,16

                                 = 800 keramik

Jumlah kardus yang diperlukan = Kebutuhan keramik : jumlah keramik dalam 1 dus

                                                    = 800 : 6

                                                    = 133,3333

Dapat dibulatkan menjadi         = 134 dus

Biaya yang dibutuhkan = jumlah kardus x harga

                                       = 134 x 65.000

                                       = 8.710.000

Biaya yang dibutuhkan untul membeli keramik merk Mulia adalah Rp. 8.710.000,00

22. sebuah kolam renang memiliki ukuran panjang 33 m, lebar 18 meter, dan kedalaman 1,5 m. Kolam renang tersebut dipasangi keramik. Dinding bagian tepi dipasang keramik warna putih. Dasar kolam renang dipasang keramik warna biru muda. Selisih luas keramik dinding bagian tepi dengan dasar kolam ada .... ​


Untuk menghitung selisih luas keramik dinding bagian tepi dan dasar kolam renang, kita perlu menghitung luas masing-masing bagian terlebih dahulu.

Luas dinding bagian tepi:

2 x (panjang + lebar) x kedalaman

2 x (33 + 18) x 1.5 = 198 m^2

Luas dasar kolam renang:

panjang x lebar x kedalaman

33 x 18 x 1.5 = 990 m^2

Selisih luas keramik dinding bagian tepi dan dasar kolam renang:

990 - 198 = 792 m^2

Jadi, selisih luas keramik dinding bagian tepi dan dasar kolam renang adalah 792 m^2.

30. Sebuah kamar mandi berukuran 2m x 1,4m yang dindingnya akan ditutup keramik setinggi 1 meter. Jika ukuran keramik 20cm x 20cm dan harganya Rp12.000/buah, maka harga keramik seluruhnya adalah?


Luas permukaan balok adalah :

= 2 x [ ( panjang x lebar ) + ( panjang x tinggi ) + ( lebar x tinggi ) ]

Sedagkan untuk mencari volume balok adalah :

= luas alas x tinggi  

= panjang x lebar x tinggi

Pembahasan

Perhatikan gambar terlampir.

Untuk menghitung jumlah keramik yg dibutuhkan, maka terlebih dahulu hitung luas permukaan balok hanya pada bagian dinding nya saja.

Luas permukaan balok pada bagian dinding sebelah kanan sama dengan luas permukaan balok pada bagian dinding sebelah kiri. Luas permukaan kedua dinding ( kanan dan kiri ), yaitu :

= 2 x panjang x lebar

= 2 x 1,4 m x 1 m  

= 2,8 m²

Luas permukaan balok pada bagian dinding sebelah depan sama dengan luas permukaan balok pada bagian dinding sebelah belakang. Luas permukaan kedua dinding ( depan dan belakang ), yaitu :

= 2 x panjang x lebar

= 2 x 2 m x 1 m  

= 4 m²

Sehingga luas dinding seluruhnya yaitu :

= 2,8 m² + 4 m²

= 6,8 m²

Hitung luas keramik, yaitu :

= sisi x sisi

= 20 cm x 20 cm

= 400 cm²

> Ubah ke dalam satuan m²

> 1 m² = 10.000 cm²

= ( 400 : 10.000 ) m²

= 0,04 m²

Menghitung banyak keramik yg di butuhkan untuk di pasang di dinding kamar mandi yaitu :

= luas permukaan dinding : luas keramik

= ( 6,8 : 0,04 ) x 1 ubin

= ([tex] \frac {68} {10}[/tex] : [tex] \frac {4} {100}[/tex] ) x 1 ubin

= ([tex] \frac {68} {10}[/tex] x [tex] \frac {100} {4}[/tex] ) x 1 ubin

= ([tex] \frac {68} {10}[/tex] x 25 ) x 1 ubin

= [tex] \frac {68 x 25} {10}[/tex] ubin

= [tex] \frac {1.700} {10}[/tex] ubin

= 170 ubin

Jika harga ubin per buah adalah Rp 12.000 maka harga keramik seluruhnya adalah :

= 170 ubin x Rp 12.000

= Rp 2.040.000
