4. Sebuah toko perlengkapan anak menyediakan sandal, sepatu, dan celana. Harga 5 pasang sandal, 2 pasang sepatu, dan 1 potong celana Rp 265.000,00. Harga 3 pasang sandal dan 1 pasang sepatu Rp 126.000,00. Harga 3 pasang sepatu dan 2 potong celana Rp 320.000,00. Di toko tersebut Bu Lia membeli 2 pasang sandal, 1 pasang sepatu , dan satu potong celana. Bu Lia harus membayar sebanyak â€¦.


Jawab:

Bu Lia harus membayar sebanyak Rp 139.000,00.

Penjelasan dengan langkah-langkah:

Bila sandal diinisialkan S, sepatu diinisialkan P dan celana diinisialkan C, maka kita dapat membuat persamaan sebagai berikut.

Persamaan (1)

5 S + 2 P + C = 265.000 ... (1)

Persamaan (2)

3 S + P = 126.000 ... (2)

Persamaan (3)

3 P + 2 C = 320.000

2 C = 320.000 - 3 P  

C = (320.000 - 3 P) / 2

C = 160.000 - 1,5 P ... (3)

Substitusikan persamaan (3) ke persamaan (1),

5 S + 2 P + C = 265.000

5 S + 2 P + (160.000 - 1,5 P) = 265.000

5 S + 0,5 P = 265.000 – 160.000

5 S + 0,5 P = 105.000, kalikan 2 untuk setiap ruas, sehingga kita dapatkan

10 S + P = 210.000 ... (4)

Eliminasi persamaan (2) dengan persamaan (4)

3 S + P = 126.000

10 S + P = 210.000

--------------------------- -

-7 S = -84.000

S = -84.000 / -7

S = 12.000

Substitusikan nilai S = 12.000 ke dalam persamaan (2),

3 S + P = 126.000

3. (12.000) + P = 126.000

36.000 + P = 126.000

P = 126.000 – 36.000

P = 90.000

Substitusikan nilai P = 90.000 ke dalam persamaan (3),

C = 160.000 - 1,5 P

C = 160.000 - 1,5 (90.000)

C = 160.000 – 135.000

C = 25.000

Maka, bila Bu Lia membeli 2 pasang sandal, 1 pasang sepatu , dan satu potong celana, nominal yang harus dibayarkan adalah

N = 2 S + P + C

N = 2. (12.000) + (90.000) + (25.000)

N = 24.000 + 115.000

N = 139.000

Di toko tersebut Bu Lia membeli 2 pasang sandal, 1 pasang sepatu , dan satu potong celana. Bu Lia harus membayar sebanyak ….


Jawab:

Diketahui :

Harga sandal = x

Harga sepatu = y

Harga celana = z

Harga 5  pasang sandal, 2 pasang sepatu, dan 1 potong celana Rp 265.000,00.

Harga 3 pasang  sandal dan 1 pasang sepatu Rp 126.000,00.

Harga 3 pasang sepatu dan 2 potong celana  Rp 320.000,00.


Ditanya :

Harga 2 pasang sandal, 1 pasang sepatu ,  dan satu potong celana = ?


Penjelasan dengan langkah-langkah:

Harga - harga tersebut diganti dengan variabel x, y, dan z untuk mempermudah perhitungan.

Perhatikan keterangan dalam soal, kemudian kita ubah dalam model matematika.

Harga 5 pasang sandal, 2 pasang sepatu, dan 1 potong celana = 265.000

Maka, 5x + 2y + z = 265.000

Harga 3 pasang  sandal dan 1 pasang sepatu = 126.000

Maka, 3x + y = 126.000

Harga 3 pasang sepatu dan 2 potong celana = 320.000

Maka, 3y + 2z = 320.000


Langkah selanjutnya, jadikan variabel x dan z pada persamaan 2 dan 3 ditnyatakan dengan nilai variabel y.

3x + y = 126.000

3x = 126.000 - y

x = (126.000 - y)/3


3y + 2z = 320.000

2z = 320.000 - 3y

z = (320.000 - 3y)/2


Setelah x dan z ditentukan dengan nilai y, masukkan ke persamaan 1.

5x + 2y + z = 265.000

5((126.000 - y)/3) + 2y + ((320.000 - 3y)/2) = 265.000 (kalikan 6 seluruhnya)
