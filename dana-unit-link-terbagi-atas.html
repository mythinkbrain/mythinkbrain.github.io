15. tolong kak kirimkan link tentang bagian-bagian lingkaran serta penjelasannya dan gambarpleaseeeeeeee soalnya dikumpulnya hari ini​


Pengertian lingkaran itu sendiri merupakan sekumpulan titik-titik pada garis lengkung yang mempuyai jarak sama terhadap satu titik di bagian tengahnya. Titik yang ada di tengah-tengah tersebut dinamakan titik pusat lingkaran. Sedangkan unsur-unsur jarak dari garis lengkung ke titik pusat dinamakan jari-jari lingkaran.

Lingkaran merupakan salah satu bangun datar, dan setiap bangun datar pasti memiliki unsur-unsur yang membentuknya. Sebuah lingkaran terdiri dari beberapa unsur bagian pembentuknya, di antaranya yaitu titik pusat, jari-jari, diameter, busur, tali busur, tembereng, juring, apotema, sudut pusat, dan sudut keliling lingkaran

Jawaban:

A. Aktif-publik di web itu maksudnya adalah semua orang bisa mengakses/ membuka file tersebut

b. aktif-siapa saja yg memiliki link maksudnya hanya orang yg diberi link file yg dapat mengaksesnya

c. nonaktif maksudnya cuman kita aja yg bisa mengakses file tsb /bersifat pribadi

Penjelasan:

semoga membantu:)

Jawaban dengan penjelasan:

a. Semua orang dapat mengakses dari searching, contohnya kamu membuat file testing.txt maka semua orang dapat mencari file tersebut di kolom pencarian web, atau searching dari website tsb

b. Semua orang yang memiliki link (tautan) saja yang bisa mengakses link tersebut, apabila orang itu tidak memiliki link (tautan), maka orang tersebut tidak bisa membuka file tsb dengan searching.

c. Semua orang tidak bisa membuka file tersebut meskipun mempunyai link atau dengan searching, dikarenakan link file tersebut sudah nonaktif, biasanya dikarenakan masa berlaku sudah habis masa.
