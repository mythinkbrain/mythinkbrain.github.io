19. Di ketahui pola bilangan 4,7,10,13,....Un, nilai dari S25+S40 adalah.....


Materi : Barisan Aritmatika


Jawaban terlampir


Semoga membantuBarisan 4,7,10,13

Un = a + (n - 1)b
Sn = n/2(2a + (n - 1)b)

U1 = a = 4
U2 = a + b = 7
b = U2 - U1
b = 7 - 4
b = 3

S25 = 25/2(2a + (25 - 1)b)
= 25/2(2a + 24b)
= 25(a + 12b)
= 25(4 + 12(3))
= 25 (4 + 36)
= 25 (40)
= 1000

S40 = 40/2(2a + (40 - 1)b)
= 20(2a + 39b)
= 20(2(4) + 39(3))
= 20 (8 + 117)
= 20 (125)
= 2500

Jadi S25 + S40 = 1000 + 2500 = 3500

Semoga membantu :)
