10. Berapa lama seseorang mengunduh file sebesar 68900mb dengan kecepatan internet 5mbps?


Bab Kecepatan
Matematika SMP Kelas VII

1 mbps = 1 mega bit per second
1 mbps = 1 mega bit per detik

68.900 mb : 5 mbps
= 13.780 seconds
= 13.780 detik
= (13.780 : 3.600) jam
= 3 jam 2.980 detik
= 3 jam 49 menit 40 detik
