10. Pak Hari baru saja membuka Toko Pakan Hewan, yang menjual pakan burung, pakan bebek, dan pakan ayam. Di hari pertama pakan burung laku 2 bungkus, pakan bebek laku 3 bungkus, dan pakan ayam laku 4 bungkus, memperoleh penghasilan Rp 22.000. Hari kedua hanya laku 1 bungkus pakan bebek dan 3 bungkus pakan ayam, memperoleh penghasilan Rp 12.000. Dan di hari ketiga, pakan burung laku 3 bungkus, pakan bebek 1 bungkus, dan pakan ayam 3 bungkus, dengan penghasilan Rp 13.600. Maka, berapa harga masing-masing pakan per bungkusnya?​


Jawaban:

misal :

pakan burung x

pakan bebek y

pakan ayam z

2x + 3y + 4z = 22.000 --> persamaan 1

0 + y + 3z = 12.000 --> persamaan 2

3x + y + 3z = 13.600 --> persamaan 3

dari persamaan 2 didapat :

y = 12.000 - 3z

substitusikan ke persamaan 1 dan 3 :

2x + 3 ( 12.000 - 3z ) + 4z = 22.000

2x + 36.000 - 9z + 4z = 22.000

2x - 5z + 12.000 = 0

3x + ( 12.000 - 3z ) + 3z = 13.600

3x + 12.000 - 3z + 3z - 13.600 = 0

3x - 1600 = 0

x = 1600/3

x = 533,33

2x - 5z + 12.000 = 0

2 ( 533,33 ) + 12.000 = 5z

1066,66 + 12.000 = 5z

13066,66 = 5z

z = 2613,33

y = 12.000 - 3z

y = 12.000 - 3 ( 2613,33 )

y = 12.000 - 7840

y = 4160

jadi masing2 harga pakan perbungkusnya:

pakan burung : 533,33 rupiah/bungkus

pakan bebek : 4160 rupiah/bungkus

pakan ayam : 2613,33 rupiah/bungkus

Neto = 40 - 0.8 = 39.2 kg /karung
Harga Beli = 80.000 × 5 = 400.000
Harga Jual = 196 × 2.500 = 490.000

B) 1 karung = 40 - 0.8 = 39.2 kg
5 karung = 39.2 × 5 = 196 kg
C) 490.000 - 400.000 = 90.000
