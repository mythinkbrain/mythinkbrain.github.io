17. Sebuah kamar berbentuk persegi panjang, panjangnya (2x+1) cm dan lebarnya (x+3) cm. Jika luas kamar tersebut 63cm², tentukan panjang dan lebar kamar tersebut?


Jawaban:

L = p x l

63 = (2x+1) (x+3)

63 = 2x² + 6x + x + 3

63 = 2x² + 7x + 3

0 = 2x² + 7x - 60

0 = (x-4) (2x+15)

maka :

x = 4 (memenuhi)

x = -15/2 (tidak memenuhi)

p = 2x+1 = 2.4+1 = 8+1 = 9

l = x+3 = 4+3 = 7

maka panjang = 9 cm dan lebar = 7 cm.

30. Bintang sedang mendesain animasi yang melibat kan kurva, f(x)=x-1 dan g(x)=5/x-3. Tentukan batas-batas nilai x yang di butuhkan bintang untuk mendesain gambar tersebut!


Jawab:

Agar kurva f(x) di atas kurva g(x) maka nilai f(x) haruslah lebih besar dari g(x).

f(x)  >  g(x)

x+1  >  5/x-3

x+1 - 5/x-3 > 0

(x+1)(x-3)/(x-3) > 0

=> pembuat nol pembilang:

   (x+1)(x-3) = 0 <=> x = -1 atau x = 3

=> pembuat nol penyebut:

   x - 3 = 0 <=> x = 3

__-__-1___+___3__-__

Jadi batas-batas nilai x yang dibutuhkan bintang untuk mendesain animasi gambar adalah  {x ∈ ℝ | -1 < x < 3}.
