10. Bagaimana Cara membedakan nomor halaman di word romawi dan angka


Jawaban:

Untuk pengguna windows

1. Pilih Insert > Page Number, lalu tentukan di bagian mana kamu mau menyisipkan halaman apakah di bagian atas, bawah, dan yang lain

2. Klik Page Number > Format Page Number, selanjutnya tinggal pilih mau format angka atau romawi

3. Misalnya kamu mau memberi halaman tapi bukan di halaman pertama melainkan halaman ketiga paper, centang kotak Different First Page

4. Klik Page Number > Format Page Number, pilih 'Start at' di Page Numbering. Kamu tinggal ketik angka 3

5. Proses selesai. Kamu bisa klik Close Header and Footer atau tekan tombol Esc.

- Membuat Halaman Romawi dan Angka di Mac

1. Pilih Insert > Page Number > Page Number, lalu tentukan posisi untuk meletakkan nomor halaman

2. Selanjutnya klik Format > Number Format, tinggal pilih apakah kamu mau halamannya berupa angka atau huruf romawi

3. Kalau ingin membuat halaman tapi bukan di halaman pertama, klik untuk hilangkan tanda ceklis di box Show number on first page

4. Lalu pilih Format, pilih 'Start at' dan masukkan di halaman berapa kamu ingin mulai ada nomor halaman

5. Lalu klik OK dan halaman akan muncul di lembar paper kamu

Nah, itu dia langkah-langkah untuk membuat halaman di Microsoft Word, Urbanreaders. Selamat mencoba!

20. Tuliskan 6 cara membuat dan mengatur nomor posisi halaman di Word 2007


★ Buat dokumen baru, posisikan kursor di lembar pertama kemudian klik Page Layout – Breaks – Next Page

★ Anda kemudian akan dilemparkan ke lembar halaman kedua, jangan pindahkan kursor, langsung saja klik Insert – Page Number – Format Page Numbers

★ Di jendela popup, pilih format penomoran 1, 2, 3 lalu klik Start at: 1 dan klik OK

★ Setelah langkah di atas dilakukan, jangan heran bila belum ada perubahan atau munculnya halaman. Klik Page Number lagi kemudian pilih posisi yang Anda inginkan, bebas

★ Nah, sekarang nomor halaman sudah muncul, tapi belum berbeda

★ Sekarang silahkan posisikan kursor ke halaman pertama, lalu klik Insert – Page Number – Format Page Numbers sekali lagi

★ Sekali lagi klik Page Number dan pilih posisi nomor halaman yang Anda inginkan lalu lihat hasilnya

22. Cara menghapus kelebihan halaman pada word


Jawaban singkat:

Misalkan ada 3 halaman di Ms. Word dan ingin menghapus halaman ketiga, caranya:

- posisikan kursor di halaman 2

- tekan tombol Delete di keyboard secara terus menerus

- bila halaman 3 sudah terhapus, hentikan menekan tombol Delete

Penjelasan tambahan:

Amatilah contohnya lampiran:

- awalnya ada 3 halaman, bila posisi kursor sudah ada di halaman 2, maka keterangan di status bar adalah PAGE 2 OF 3 (ditunjukkan tanda panah merah)

- tekan tombol Delete di keyboard terus menerus hingga tampilan selanjutnya ditunjukkan tanda panah biru

- bila halaman 3 sudah terhapus, maka keterangan di status bar menjadi PAGE 2 OF 2 (ditunjukkan tanda panah hijau)

dihalaman yang berlebih cukup blok halaman tersebut lalu klik tombol "del" atau "delete" setelah itu halaman tersebut akan secara otomatis terhapus.
