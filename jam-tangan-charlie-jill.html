1. 9. Alice dan Laura dapat menyelesaikan sebuah pekerjaan dalam 2 jam. Alice dan Charlie dapat menyelesaikan pekerjaan yang sama dalam waktu 3 jam. Laura dan Charlie dapat menyelesaikan pekerjaan tersebut dalam 4 jam. Berapa lama pekerjaan selesai jika Alice, Laura, dan Charlie bekerja bersama?​


Berapa lama pekerjaan selesai jika Alice, Laura, dan Charlie bekerja bersama?

[tex]1 \frac{11}{13} jam[/tex]

pembahasan :

Alice = A

Laura = L

Charlie = C

perbandingan berbalik nilai.

1/A + 1/L = 1/2

1/A + 1/C = 1/3

1/L + 1/C = 1/4

setelah dieliminasi/substitusi didapat:

A = 24/7 , C = 24, L = 24/5

jika Alice, Laura, dan Charlie bekerja bersama = X

1/A + 1/C + 1/L = 1/X

1/(24/7) + 1/24 + 1/(24/5) = 1/X

7/24 + 1/24 + 5/24 = 1/X

13/24 = 1/X

X = 24/13

X =

[tex]1 \frac{11}{13} [/tex]

jam
