1. 50 cabang ilmu biologi dan pengertiannya​


50 cabang biologi :

Acarologi : Wawasan tungau. Contoh : cara tungau menghisap darah.

Aerobiologi : Wawasan bakteri-bakteri organik yang disebarkan udara. Contoh : jenis-jenis virus.

Agroforestri : Wawasan pengolahan pertanian. Contoh : cara melakukan persilangan tanaman.

Agronomi : Wawasan membudidayakan tanaman. Contoh : cara memilih bibit.

Algologi : Wawasan alga. Contoh : cara alga berkembangbiak.

Anatomi : Wawasan bagian-bagian tubuh. Contoh : fungsi dari tubuh.

Anatomi perbandingan : Wawasan perbandingan anatomi makhluk hidup. Contoh : perbandingan kucing dan harimau.

Andrologi : Wawasan reproduksi pria. Contoh : penyakit pada reproduksi pria.

Anestesiologi : Wawasan anestesi. Contoh : cara penggunaan anestesi.

Angiologi : Wawasan peredaran darah dan limfa. Contoh : penyakit pada peredaran darah.

Antropologi biologi : Wawasan penelitian biologis dan evolusi. Contoh : sejarah evolusi manusia.

Apiologi: Wawasan lebah. Contoh : ternak lebah

Arachnologi : Wawasan laba-laba. Contoh : jenis laba-laba.

Artrolog : Wawasan sendi. Contoh : penyakit sendi.

Artropodologi: Wawasan hewan artropoda. Contoh : ciri artropoda.

Astrobiologi : Wawasan kemungkinan biologi masa depan. Contoh : pengembangan evolusi hewan.

Bakteriologi : Wawasan bakteri. Contoh : macam-macam bakteri.

Batrakologi : Wawasan amfibi. Contoh : hewan-hewan amfibi.

Biofarmaka : Wawasan tanaman obat. Contoh : budidaya tanaman obat.

Bioinformatika : Wawasan komputer dalam menggali ilmu biologi. Contoh : pendalaman sel melalui komputer.

Biologi air tawar : Wawasan habitat air tawar. Contoh : hewan di air tawar.

Biologi evolusioner : Wawasan asal-usul dan evolusi makhluk hidup. Contoh : proses evolusi buaya.

Biologi integratif : Wawasan makhluk hidup secara lengkap. Contoh : definisi sel dalam ilmu biologi, fisika dan kimia.

Biologi kelautan: Wawasan kehidupan di laut. Contoh : rantai makanan di laut.

Biologi konservasi : Wawasan pelestarian alam. Contoh : pelestarian in situ dan ex situ.

Biologi kuantum : Wawasan kuantum dengan biologis. Contoh proses fotosintesis.

Biologi lingkungan : Wawasan lingkungan. Contoh : Pelestarian lingkungan.

Biologi molekuler : Wawasan terkait molekul. Contoh : pembuatan DNA.

Biologi pembangunan : Wawasan lingkungan hidup dalam ruangan. Contoh : kehidupan tanaman dalam ruangan.

Biologi perkembangan : Wawasan pertumbuhan dan perkembangan organisme. Contoh : proses pertumbuhan.

Biologi populasi : Wawasan populasi organisme. Contoh : pertahanan keberadaan populasi.

Biologi psikologi : Wawasan hubungan psikologis dan biologis. Contoh : pengaruh psikologi manusia pada fungsi tubuh.

Biologi reproduksi : Wawasan perkembangbiakan. Contoh : cara berkembang biak.

Biologi sintesis : Wawasan pembentukan biologis baru. Contoh : pengkodean RNA.

Biokimia : Wawasan kimia makhluk hidup. Contoh : penyusun enzim.

Biofisika : Wawasan hukum fisika pada objek biologis. Contoh : tekanan darah.

Biogeografi : Wawasan keanekaragaman hayati atau biologi. Contoh : jenis-jenis hutan.

Biomatematika : Wawasan biologi secara kuantitas. Contoh : peningkatan atau penurunan energi pada rantai makanan.

Biomekanika : Wawasan mekanis dalam objek biologis. Contoh : gerak termasuk dalam hukum newton.

Bionik : Wawasan pengembangan alat elektronik menjadi anatomi. Contoh : pembuatan tangan palsu.

Biostatistika : Wawasan statistika terhadap objek biologi. Contoh : percobaan biologi.

Bioteknologi: Wawasan manfaat makhluk hidup dalam proses industri. Contoh : pembuatan yogurt.

Botani : Wawasan seputar tumbuhan. Contoh : anatomi tumbuhan.

Bryologi : Wawasan bryophyta. Contoh : ciri-ciri bryophyta.

Cetologi : Wawasan cetacean. Contoh : ekosistem paus.

Conchologi : Wawasan moluska. Contoh : susunan kulit moluska.

Dendrokronologi : Wawasan usia pohon. Contoh : menentukan usia pohon dari potongan melintang batang.

Dendrologi : Wawasan pohon dan tanaman berkayu. Contoh : anatomi dasar pada pohon.

Dermatologi : Wawasan tentang kulit. Contoh : penyakit pada kulit.

Ekofisiologi : Wawasan fisik makhluk hidup. Contoh : bentuk fisik ular.
