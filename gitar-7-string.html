15. 7. Jika diketahui 3 variabel (nama, Nim, dan Alamat) dengan tipe datanya adalah sebagai berikut : O Nama: string [50] Nim: string [10] Alamat : string [200] Jika ketiga variabel tersebut diberikan data sebanyak 1000, tentukanlah berapa kapasitas memori yang terpakai untuk menampung seluruh variabel tersebut. ​


Kapasitas memori yang terpakai untuk menampung seluruh variabel tersebut adalah 263.000 Byte, atau sekitar 256,836 KB.
 

Pembahasan

Terlepas dari bahasa pemrograman dan lingkungan pengembangan yang digunakan, berdasarkan definisi, tipe data [tex]\tt char[/tex] membutuhkan 1 byte memori. Sedangkan tipe data [tex]\tt string[/tex] adalah rangkaian/array dari [tex]\tt char[/tex], ditambah 1 karakter terminasi (null), yaitu ’\0’.

Sebagai contoh: string [tex]\tt "brainly"[/tex] merupakan rangkaian dari karakter-karakter [tex]\tt ['b',\:'r',\:'a',\:'i',\:'n',\:'l',\:'y',\:'\backslash0'][/tex].

Jadi, jika sebuah variabel dideklarasikan bertipe data [tex]\tt string[n][/tex] dengan n menyatakan panjang maksimum variabel tersebut (bisa berbeda nama tipe datanya tergantung bahasa pemrograman atau lingkungan pengembangan), maka variabel tersebut membutuhkan [tex]n+1[/tex] Byte pada memori.

Diberikan tiga variabel

[tex]\tt Nama: string [50][/tex],[tex]\tt Nim: string [10][/tex], dan[tex]\tt Alamat : string [200][/tex]

Jika ketiga variabel tersebut diberikan data sebanyak 1000, maka secara teoritis, kapasitas memori yang terpakai atau dibutuhkan adalah:
[(50+1)+(10+1)+(200+1)] Byte × 1000
= (263 × 1000) Byte
= 263.000 Byte
  → ( 1 KB = 2^10 Byte = 1.024 Byte)
= (263.000/1.024) KB
≈ 256,836 KB

Jawaban:

11 Hal Tentang String Pada Java Yang Perlu Diketahui

Banyak hal yang bisa dipelajari oleh pemula dalam Java mengenai String, sehingga dengan memiliki pengetahuan mendalam mengenai method. String yang berbeda akan membuat anda bisa menggunakannya dengan tepat. String dalam Java adalah class yang sangat spesial dan sering sekali digunakan. Dikarenakan banyak digunakan pada berbagai project pada Java, maka di sini akan di kupas mengenai 11 hal yang wajib diketahui mengenai String.

Java String

Penjelasan Tentang String

Apa sajakah yang penting diketahui mengenai String?

Di bawah ini adalah penjelasan mengenai String :

1. Cara membuat objek String pada java ada 2 cara.

Untuk membuat objek String dapat dilakukan dengan dua cara, yaitu:

String a = “halo”; //membuat objek String dengan String literal

String a = new String(“halo”); //membuat objek String dengan operator new

2. String bersifat Immutable dan final

String bersifat immutable, artinya sekali sudah dibuat, maka kita tidak dapat memodifikasi isi dari String tersebut. Jika kita memodifikasi String dengan method toLowerCase(), toUpperCase() ataupun method lainnya, maka akan selalu menghasilkan String yang baru. Dikarenakan String adalah final, maka tidak ada jalan siapapun untuk melakukan extend String atau meng-override fungsional apapun dari String.

3. String diatur dalam String pool

Setiap kali kita membuat objek String dengan menggunakan String literal, maka Java Virtual Machine (JVM) pertama kali akan memeriksa konten dari objek yang dibuat. Jika terdapat objek yang sama dalam String pool yang memiliki konten yang sama, maka referensi akan dikembalikan pada objek tersebut, tanpa membuat objek baru. JVM tidak melakukan pemeriksaan String pool jika kita membuat objek dengan menggunakan operator new.

Untuk lebih jelasnya :

Java

"objek String yang sama? " + a);

boolean b = nama==nama_2;

System.out.println("Apakah nama dan nama-2 mengacu pada " +

}

}

4. String adalah objek

String merupakan sebuah objek, bukanlah seperti tipe data primitif yang lain seperti int, double dan sebagainya.

5. Gunakan method equals () untuk membandingkan String

Untuk membandingkan String, anda dapat menggunakan method equals().Class String meng-override method equals() dan memberikan konten equality dari String, yang berdasarkan pada character dan urutan. Jadi bila anda ingin membandingkan dua objek String, dan akan memeriksa apakah String tersebut sama atau tidak, selalu gunakan method equals() daripada menggunakan operator equality (= =).

30. Maestro gitar Belawan memainkan musik gitar dengan senar yang terbuat dari bahan nilon dan bahan string. Panjang senar nilon 9 kali panjang senar string, gaya tekan senar nilon 81 kali dari gaya tekan senar string, luas pemampang senar strin 16 kali dari luas pemampang senar nilon, dan massa jenis senar string sama dengan massa jenis senar nilon. Tentukan perbandingan frekuensi antar senar dari bahan nilon dengan bahan string !


BUNYI
• Sumber Bunyi
— senar/dawai

Indeks ₙ untuk nilon dan ₛ untuk string

Lₙ = 9 Lₛ
Fₙ = 81 Fₛ
Aₛ = 16 Aₙ
ρₙ = ρₛ
f₀ₙ : f₀ₛ = __?

Frekuensi nada pada senar
fₙ = (n + 1) / (2 L) × √(F/(ρ A)

Frekuensi nada dasar → n = 1
f₀ = 1 / (2 L) × √(F/(ρ A)

perbandingan frekuensi nada dasar
f₀ₙ / f₀ₛ = (Lₛ/Lₙ) × √( (Fₙ/Fₛ) × (Aₛ/Aₙ) )
f₀ₙ / f₀ₛ = 1/9 × √(81 × 16)
f₀ₙ / f₀ₛ = 1/9 × 9 × 4
f₀ₙ / f₀ₛ = 4
f₀ₙ / f₀ₛ = 4 : 1 ✔️
