29. Membuka folder dapat dilakukan dengancara mengeklik double di folder tersebut,ataud ngan mengeklik tanda plus (+) yangterdapat di depan ....a. duta tujuanb. ikon folderc. membuka folderd. posisi semulae. sesuai jumlah dokumen​


Membuka folder dengan cara mengeklik double folder maupun dengan tanda plus terdapat didepan B. Ikon folder

Ikon merupakan lambang atau simbol aplikasi yang ada di desktop yang berfungsi untuk mempermudah mengenali aplikasi yang akan digunakan atau sering digunakan. Pada setiap aplikasi pada umumnya memiliki ikon yang berbeda-beda sehingga mudah untuk mengenalinya jika akan digunakan.

Pembahasan

Folder yang ada di windows berfungsi untuk menyimpan file. Ikon folder pada windows memiliki simbol gambar map atau amplop kecil yang memiliki warna coklat. Ikon dapat dibuat sesuai dengan keinginan agar tidak bosan ketika kita melihatnya. Namun kebanyakan orang pengguna laptop atau komputer lainnya menerima apa adanya ikon folder tanpa ada keinginan untuk menggantinya.

Berikut ini cara yang dapat digunakan untuk mengganti ikon folder :

Nyalakan laptop atau komputer yang akan diubah ikon foldernya.Buka file explorer, kemudian klik kanan untuk membuka jendela pintasan dan pilih properties.Setelah klik properties pastikan berada pada tab customize, kemudian klik change ikon.Setelah muncul daftar ikon, kemudian tekan tombol ok untuk menggunakan ikon tersebut.Pilih ikon folder yang akan diterapkan pada laptop atau komputer, kemudian kembali pada change ikon sesuai nama folder yang akan diganti dan klik tombol apply.
