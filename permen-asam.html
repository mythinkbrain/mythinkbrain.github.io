jumlah permen gula = (8/(8+4))x
jumlah permen asam = (4/(8+4))x
Selisih nya adalah 50, maka
(8/12)x - (4/12)x = 50
(4/12)x = 50
x = (50 x 12)/4
x = 600/4 = 150
jadi jumlah seluruh permen adalah 150

11. Suatu permen karet mengandung 2,5% urea NH2CONH2. Urea tersebut dapat bereaksi dengan asam cuka sesuai persamaa NH2CONH2+2CH3CO2H=CO2+2CH3CO2-NH3 Tentukan berapa massa permen karet untuk menetralkan 1,00 gram asam cuka


[Urea] dalam permen karet : 2,5% (b/b)
massa asam cuka : 1,00 gram
massa permen karet : ..... g ?

jawab

langkah 1
Tuliskan reaksi terlebih dahulu. Reaksi sudah dibuatkan jadi tinggal perhatikan dengan baik baik.

Reaksi :
NH2CONH2+2CH3CO2H=CO2+2CH3CO2-NH3

langkah 2
Tentukan mol asam cuka

asam cuka → CH3COOH

mol : massa / Mr

Mr CH3COOH : 12(2) + 1(4) + 16(2)
Mr CH3COOH : 60 g/mol

mol CH3COOH : 1 g / 60 g/mol
CH3COOH : 0,0167 mol

langkah 3
Tentukan mol urea. Caranya bandingkan koefisien reaksi (K) urea terhadap koefisien reaksi (K) asam cuka yg telah diketahui molnya

mol urea : (K. urea / K. as. cuka) x mol as. cuka
mol urea : (1/2) x 0,0167 mol
urea : 0,0084 mol

langkah 4
Tentukan massa urea.

Mr urea : 14(2) + 1(4) + 12 + 16
Mr urea : 60 g/mol

mol : massa / Mr
massa : mol x Mr

massa urea : 0,0084 mol x 60 g/mol
urea : 0,504 gram

langkah 5
Tentukan massa permen karet

% (b/b) : (m. urea/ m. permen karet) x 100%

2,5% : (0,504 g / m. permen karet) x 100%
2,5 (g/g) x m. permen karet : 0,504 g x 100
2,5m. permen karet (g/g) : 50,4 g
massa permen karet : 50,4 g / 2,5 (g/g)
massa permen karet : 20,16 gram

Jadi, massa permen karet yg mengandung 2,5% urea sebesar 20,16 gram
