23. apakah satuan yang digunakan untuk mengukur kecepatan transfer data di internet


satuan untuk mengukur kecepatan di internet ialah Kbps ( kilobits per second ) Dan Mbps ( megabits per second ) .
1 Kbps = 1.000 bits & 1 Mbps = 1.000.000 bits.

Dijawab ya................

Bps = Bites per second = 1bps

Kbps = Kilo Bites per second = 1000 bps/10^3 bps

Mbps = Mega Bites per second = 1.000 kbps/1.000.000bps/10^6

Gbps = Giga Bites per second = 1 Gbps = 1.000.000.000 bps = 10^9 bps = 1.000 Mbps

Tbps = Terra Bites per second = 1 Tbps = 1.000 Gbps
