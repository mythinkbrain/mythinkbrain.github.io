⟩ SPLTV

Misalkan :

buku = xspidol = ytinta = z

Bentuk persamaan :

3x + y + 2z = 17.200 ...(1)

2x + 2y + 3z = 19.700 ....(2)

x + 2y + 2z = 14.000 ....(3)

Ubah bentuk pers. (1) :

3x + y + 2z = 17.200

y + 2z - 2z = 17.200 - 3x - 2z

y = 17.200 - 3x - 2z ....(4)

Substitusikan ke pers. (2) dan (3) :

2x + 2(17.200 - 3x - 2z) + 3z = 19.700

x + 2(17.200 - 3x - 2z) + 2z = 14.000

Jabarkan/Sederhanakan :

2x + 2(17.200 - 3x - 2z) + 3z = 19.700

2x + 34.400 - 6x - 4z + 3z = 19.700

-4x + 34.400 - z = 19.700

-4x - z = 19.700 - 34.400

-4x - z = -14.700 ...(5)

-2(-4x - z) = -2(-14.700)

8x + 2z = 29.400 ...(6)

x + 2(17.200 - 3x - 2z) + 2z = 14.000

x + 34.400 - 6x - 4z + 2z = 14.000

-5x + 34.400 - 2z = 14.000

-5x - 2z = 14.000 - 34.400

-5x - 2z = -20.400 ....(7)

Masukkan pers.. (7) ke pers. (6)

8x + 2z - 5x - 2z = 29.400 - 20.400

8x - 0 - 5x = 29.400 - 20.400

3x = 29.400 - 20.400

3x = 9.000

x = 9.000 ÷ 3

x = 3.000,-

Substitusikan nilai x ke pers. (5)

-4x - z = -14.700

-4(3.000) - z = -14.700

-12.000 - z = -14.700

-z = -14.700 + 12.000

-z = -2.700

z = 2.700,-

Substitusikan nilai z dan x ke pers. (4)

y = 17.200 - 3x - 2z

y = 17.200 - 3(3.000) - 2(2.700)

y = 17.200 - 9.000 - 5.400

y = 2.800,-

Harga 1 buah buku adalah Rp3.000, 1 buah spidol adalah Rp2.800, dan 1 buah tinta adalah Rp2.700.

----------------------------------------------------------------------

Pendahuluan

SPLTV atau Sistem Persamaan Linear Tiga Variabel merupakan suatu persamaan yang dimana terdapat kalimat terbuka yang menerapkan relasi tanda sama dengan ( = ) dan memiliki tiga persamaan linear yang tiap persamaannya memiliki tiga peubah/variabel dengan masing-masing berpangkat satu.

Metode-metode yang digunakan dalam mengerjakan Persamaan Linear Tiga Variabel.

Metode Substitusi, yaitu menggantikan (substitusi) salah satu peubah (variabel) persamaan dari peubah persamaan lainnya.Metode Eliminasi, yaitu menghilangkan (eliminasi) peubah (variabel) persamaan dengan cara mengurangkan atau menambahkan persamaan yang ada.Metode Campuran, yaitu pengerjaan yang dilakukan dengan menggabungankan cara substitusi dan eliminasi.

» Eliminasi “variabel z” dari pers (i) dan pers (iii)

3x + y + 2z = 17.200

ㅤㅤ2x – y = 3.200

ㅤㅤ ㅤㅤ y = 2x – 3.200 … pers (v)

⏣ Metode Substitusi

» Substitusikan pers (v) ke pers (iv)

5x – y = 12.200

5x – (2x – 3.200) = 12.200

5x – 2x + 3.200 = 12.200

3x + 3.200 = 12.200

3x = 12.200 – 3.200

3x = 9.000

x = 9.000 ÷ 3

x = 3.000 ✅

» Substitusikan “nilai x” ke pers (v)

y = 2x – 3.200

y = 2(3.000) – 3.200

y = 6.000 – 3.200

y = 2.800 ✅

» Substitusikan “nilai x dan y” ke pers (iii)

x + 2y + 2z = 14.000

3.000 + 2(2.800) + 2z = 14.000

3.000 + 5.600 + 2z = 14.000

8.600 + 2z = 14.000

2z = 14.000 – 8.600

2z = 5.400

z = 5.400 ÷ 2

z = 2.700 ✅
